# Resumen del Blueprint

Todos los nodos diseñados para blueprints están contenidos dentro de la clase `UThirdwebFunctionLibrary`. Proporciona un conjunto completo de
funciones de Blueprint para interactuar con las funcionalidades de billetera de Thirdweb dentro de Unreal Engine. Esto incluye utilidades
para la creación de billeteras, gestión de sesiones, firma de mensajes y más.

## Notas de Implementación Interna

- El prefijo de función `BP_` es un estándar de Unreal Engine diseñado para asegurar que los desarrolladores de C++ no utilicen funciones destinadas
  al uso de Blueprints.
- El prefijo de función `Conv_` es una convención requerida por Unreal Engine para detectar nodos de autocasting.
- Todos los prefijos de las funciones de operadores son una convención de Unreal Engine necesaria para detectar los comodines de nodos operadores estándar.
