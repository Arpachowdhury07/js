# ThirdwebUtils

El namespace `ThirdwebUtils` proporciona un conjunto de funciones de utilidad para uso independiente de clases.

## Funciones de Validación
- `IsChecksummedAddress(Address)`: Verifica si la dirección de Ethereum proporcionada tiene suma de control.
- `IsValidAddress(Address)`: Verifica si la dirección de Ethereum proporcionada es válida.
- `IsValidPrivateKey(PrivateKey)`: Determina si la cadena proporcionada es una clave privada válida.
- `ToChecksummedAddress(Address)`: Convierte la dirección de Ethereum proporcionada a una dirección con suma de control.

## Funciones de Conversión
- `ToText(Provider)`: Convierte un valor del enum [EThirdwebOAuthProvider][provider-enum] a su representación correspondiente en FText.
- `ToString(Provider)`: Convierte un valor del enum [EThirdwebOAuthProvider][provider-enum] a su representación en cadena.
