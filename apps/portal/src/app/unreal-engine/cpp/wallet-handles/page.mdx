# WalletHandle

`FWalletHandle` es la estructura base que proporciona un manejador único para distinguir diferentes tipos de billeteras en el
SDK de Thirdweb. Soporta una variedad de operaciones compartidas y no se usa directamente, sino que se hereda.

## Enums

### EWalletHandleType
Identificador del tipo de manejador de billetera.
- `InvalidHandle`: Representa un manejador no inicializado o inválido.
- `InApp`: Representa una billetera in-app.
- `Smart`: Representa una billetera de contrato inteligente.

## Funciones Miembro

- **`IsValid()`**: Verifica si el manejador de billetera es válido.
- **`Invalidate()`**: Invalida el manejador de billetera, configurando su ID a cero y marcando su tipo como inválido.
- **`ToAddress()`**: Recupera la dirección de billetera asociada a este manejador.
- **`Sign(Message)`**: Firma un mensaje utilizando el manejador de billetera.
- **`GetType()`**: Obtiene el tipo de manejador de billetera.
- **`GetID()`**: Obtiene el ID del manejador de billetera.
- **`GetDisplayName()`**: Recupera el nombre de depuración del manejador de billetera.
- **`GetTypeString()`**: Recupera el tipo del manejador de billetera como una cadena.

## Operadores

- **`operator==`**: Compara dos manejadores de billetera para verificar si son iguales.
- **`operator!=`**: Compara dos manejadores de billetera para verificar si son diferentes.

# InAppWalletHandle

`FInAppWalletHandle` proporciona un manejador único para billeteras InApp. Soporta una variedad de operaciones para autenticación.

## Enums

### EInAppSource
Fuente que creó la billetera InApp
- `InvalidSource`: Representa una fuente no inicializada o inválida.
- `OAuthProvider`: Representa una fuente de proveedor OAuth.
- `Email`: Representa una fuente de correo electrónico.
- `Phone`: Representa una fuente de teléfono.
- `Jwt`: Representa una fuente JWT.
- `AuthEndpoint`: Representa una fuente de endpoint de autenticación.
- `Guest`: Representa una fuente de invitado.

## Funciones Miembro

- **`CreateEmailWallet(Email, Wallet, Error)`**: Crea una billetera in-app de correo electrónico.
- **`CreateEcosystemEmailWallet(PartnerId, Email, Wallet, Error)`**: Crea una billetera in-app de correo electrónico para ecosistema.
- **`CreateOAuthWallet(Provider, Wallet, Error)`**: Crea una billetera in-app OAuth utilizando el proveedor especificado.
- **`CreateEcosystemOAuthWallet(PartnerId, Provider, Wallet, Error)`**: Crea una billetera in-app OAuth para ecosistema utilizando el proveedor especificado.
- **`CreatePhoneWallet(Phone, Wallet, Error)`**: Crea una billetera in-app de teléfono.
- **`CreateEcosystemPhoneWallet(PartnerId, Phone, Wallet, Error)`**: Crea una billetera in-app de teléfono para ecosistema.
- **`CreateJwtWallet(Wallet, Error)`**: Crea una billetera in-app JWT.
- **`CreateEcosystemJwtWallet(PartnerId, Wallet, Error)`**: Crea una billetera JWT para ecosistema.
- **`CreateAuthEndpointWallet(Wallet, Error)`**: Crea una billetera in-app utilizando un endpoint de autenticación.
- **`CreateEcosystemAuthEndpointWallet(PartnerId, Wallet, Error)`**: Crea una billetera para ecosistema utilizando un endpoint de autenticación.
- **`CreateGuestWallet(Email, Wallet, Error)`**: Crea una billetera in-app para invitados.
- **`CreateEcosystemGuestWallet(PartnerId, Email, Wallet, Error)`**: Crea una billetera para invitados en ecosistema.
- **`IsConnected()`**: Verifica si la billetera está conectada a una sesión.
- **`Disconnect()`**: Desconecta una billetera de una sesión.
- **`VerifyOTP(Method, OTP, Error)`**: Verifica un OTP para billeteras in-app.
- **`SendOTP(Method, Error)`**: Envía un OTP para billeteras in-app.
- **`FetchOAuthLoginURL(RedirectUrl, LoginLink, Error)`**: Recupera la URL de login OAuth para billeteras in-app.
- **`SignInWithOAuth(AuthResult, Error)`**: Inicia sesión utilizando los detalles de autenticación OAuth.
- **`SignInWithJwt(Jwt, Error)`**: Inicia sesión utilizando un JWT.
- **`SignInWithAuthEndpoint(Jwt, Error)`**: Inicia sesión utilizando cualquier payload.
- **`SignInWithGuest(Error)`**: Inicia sesión como usuario invitado utilizando una huella digital semi-reproducible del dispositivo. Las huellas pueden cambiar con cada actualización y no deben ser usadas a largo plazo.
- **`GetSourceString()`**: Recupera la fuente como un `FString`.
- **`GetSourceString(Source)`**: Recupera la fuente especificada como un `FString`.
- **`GetOAuthProvider()`**: Recupera el proveedor OAuth configurado.

# SmartWalletHandle

`FSmartWalletHandle` proporciona un manejador único para billeteras inteligentes. Soporta una variedad de operaciones para gestionar
sesiones e interactuar con billeteras de contratos inteligentes.

## Funciones Miembro

- **`Create(InInAppWallet, ChainID, bGasless, Factory, AccountOverride, bSuccess, Error)`**: Crea una billetera inteligente.
- **`IsDeployed(bDeployed, Error)`**: Verifica si la billetera inteligente está desplegada.
- **`CreateSessionKey(Signer, ApprovedTargets, NativeTokenLimitPerTransactionInWei, PermissionStart, PermissionEnd,
RequestValidityStart, RequestValidityEnd, TransactionHash, Error)`**: Crea una clave de sesión para una billetera inteligente.
- **`RevokeSessionKey(Signer, Error)`**: Revoca una clave de sesión para una billetera inteligente.
- **`GetAdmins(Admins, Error)`**: Recupera los administradores de una billetera inteligente.
- **`AddAdmin(Signer, Error)`**: Agrega un administrador a una billetera inteligente.
- **`RemoveAdmin(Signer, Error)`**: Elimina un administrador de una billetera inteligente.
- **`GetActiveSigners(Signers, Error)`**: Recupera los firmantes activos de una billetera inteligente.
