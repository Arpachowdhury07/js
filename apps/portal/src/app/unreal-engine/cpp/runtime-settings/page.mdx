# Configuración de Runtime

`ThirdwebRuntimeSettings` proporciona configuraciones para el SDK de Thirdweb dentro de Unreal Engine. Estas configuraciones son utilizadas por el SDK para gestionar información esencial como credenciales de cliente y rutas de almacenamiento.

Las configuraciones de runtime se encuentran en `Editar > Configuración del Proyecto > Engine > Thirdweb`. También puedes acceder a la configuración haciendo clic en el logo de thirdweb en la barra de acciones de tu editor, o manualmente en `DefaultEngine.ini`.

## **Propiedades**

- `ClientID`: El client ID utilizado para autenticarse con los servicios de Thirdweb.
- `BundleID`: El ID del paquete asociado con la aplicación.
- `EncryptionKey`: Requerido si se utilizan métodos de autenticación personalizados a través de billeteras InApp estándar (No Ecosistema).
- `EngineSigners`: Array opcional de firmantes del engine almacenados globalmente para conveniencia.
- `EcosystemId`: Identificador de la billetera de Ecosistema vinculada a tu cuenta de Ecosistema Thirdweb.
  Solo relevante cuando se utilizan Billeteras de Ecosistema. Ejemplo: `ecosystem.my-cool-game`.
- `bSendAnalytics`: Opción para habilitar o deshabilitar las analíticas de conexión.
- `bOverrideOAuthBrowserProviderBackends`: Condición de edición para sobrescribir los backends de proveedores OAuth de navegador.
- `OAuthBrowserProviderBackendOverrides`: Array de métodos de backend de proveedores OAuth que pueden ser sobrescritos.

### Requisitos de Configuración

`ClientID` + `BundleID` son necesarios para la autenticación con los servicios de Thirdweb.
`EcosystemId` es requerido si se utilizan billeteras de ecosistema.

### Funciones de Clase

- **`Get()`**: Recupera la instancia actual de `UThirdwebRuntimeSettings`.
- **`GetThirdwebGlobalEngineSigners()`**: Recupera el array de `EngineSigners`.
- **`GetThirdwebGlobalEngineSigner(bFound)`**: Recupera el primer firmante del engine del array de `EngineSigners`, si existe.
- **`GetEncryptionKey()`**: Accesor estático para obtener `EncryptionKey`.
- **`GetStorageDirectory()`**: Accesor estático para recuperar la ruta absoluta del directorio de almacenamiento de la plataforma InAppWallet de thirdweb.
- **`IsExternalOAuthBackend(Provider)`**: Accesor estático para obtener el backend resuelto de un proveedor OAuth.
- **`GetEcosystemId()`**: Accesor estático para obtener `EcosystemId`.
- **`IsEcosystem()`**: Accesor estático para verificar la validez de `EcosystemId`.
- **`GetClientId()`**: Accesor estático para obtener `ClientID`.
- **`GetBundleId()`**: Accesor estático para obtener `BundleId`.
- **`AnalyticsEnabled()`**: Accesor estático para verificar el estado de opt-in de las analíticas.
