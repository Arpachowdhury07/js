# Unreal Engine SDK

Un plugin de código para Unreal Engine que permite a los desarrolladores crear billeteras In-App, Ecosistema y
Smart Wallets para sus juegos y aplicaciones. El plugin proporciona una API sencilla y blueprints para interactuar con
billeteras, iniciar sesión con correo electrónico o redes sociales, así como crear claves de sesión para Smart Wallets.

Con este plugin, puedes mantener la incorporación de tus usuarios dentro de la aplicación y otorgar una clave de sesión a tu
[backend impulsado por thirdweb Engine][engine] para interactuar con la blockchain,
sin necesidad de firmas.

El Plugin de Unreal de Thirdweb se distribuye como un [Unreal Marketplace Code Plugin][ue-marketplace].

## Características

- **Crear e Iniciar Sesión en Billeteras In-App** - habilitando la autenticación por correo electrónico y redes sociales para crear billeteras incrustadas persistentes,
  multiplataforma, multi-dispositivo y sin necesidad de firmas.
- **Desbloquear el poder de la Abstracción de Cuentas** - desbloquea la capacidad de patrocinar gas e interactuar en nombre de los usuarios al
  actualizar cualquier billetera a una Smart Wallet, y usando Claves de Sesión para otorgar acceso limitado a tu backend - completamente
  compatible con [thirdweb Engine](https://portal.thirdweb.com/engine) u otros SDKs.
- **Core en Rust** - la lógica principal del plugin está construida en Rust, lo que lo hace extremadamente rápido y minimiza el desorden en
  tu proyecto de Unreal.
- **Compatible con Unreal Engine 5.3-5.5** - el plugin es compatible con las últimas versiones de Unreal Engine,
  permitiéndote aprovechar las nuevas características y mejoras.

## Instalación

Descarga e instala el Plugin de Unreal de Thirdweb desde [Unreal Marketplace][ue-marketplace].

Nuestro plugin está codificado utilizando un sistema de manejo intuitivo y tareas asíncronas en blueprint, lo que te permite interactuar con él de forma muy
sencilla.
