# Guía de Inicio Rápido

<Callout variant='info' title="Se requiere una clave de API">
Para comenzar con el SDK de Unreal, necesitarás tener una [clave de API de thirdweb][api-key] creada. Asegúrate de
permitir un ID de paquete para usar dentro de aplicaciones nativas de Unreal.
</Callout>

## Pasos para la Integración

<Steps>

<Step title="Descarga del Plugin" >
Descarga e instala el Plugin de Unreal de Thirdweb desde el [Unreal Marketplace][marketplace].
</Step>

<Step title="Configuración de Runtime" >
Una vez que importes el plugin a tu proyecto, deberás configurar los ajustes del plugin con tu Client ID y
Bundle ID.

Puedes hacerlo haciendo clic en el icono de thirdweb en la barra de acciones de tu engine, o navegando a
`Editar > Configuración del Proyecto > Engine > Thirdweb` y completando los campos requeridos.

<DocImage src={unrealSettings} className="w-[800px]" />

Alternativamente, puedes ir a la carpeta raíz `Config` y abrir el archivo `DefaultEngine.ini` para configurar los ajustes del plugin.

```ini
[/Script/Thirdweb.ThirdwebRuntimeSettings]
ClientID=
BundleID=
```

</Step>

<Step title="¡Pruébalo!" >
Prueba nuestro ejemplo preconstruido `Level_Thirdweb` para crear e iniciar sesión en billeteras sin problemas, utilizando todo tipo de
autenticación, y explora casos avanzados de Abstracción de Cuentas.
<DocImage src={unrealLevel} className="w-[800px]" />
</Step>

</Steps>
