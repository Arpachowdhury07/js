# Seguridad

## Manejo de datos

- Dado que Engine es autohospedado, tú (el desarrollador) mantienes el control sobre el servidor, la base de datos, los registros y la observabilidad.
- Engine maneja los datos de configuración cifrados tanto en tránsito como en reposo.
- Los firmantes de billeteras backend se reconstruyen solo en memoria en tu instancia de Engine, y estos datos nunca se envían a thirdweb ni a otras plataformas externas.
- thirdweb puede recopilar la siguiente información:
  - Métricas sobre qué cuentas están utilizando Engine
  - Métricas anonimizadas sobre el uso
  - Historial de transacciones
  - (TBD futuros datos para potenciar análisis avanzados)

## Auditoría de seguridad de terceros

A partir de enero de 2024, Engine ha recibido una auditoría de seguridad exhaustiva y un programa interno de recompensas por errores de HackerOne. Se puede proporcionar una Carta de Atestación a petición.

## Divulgación responsable

Para reportar una vulnerabilidad de seguridad, por favor contacta a security@thirdweb.com.
