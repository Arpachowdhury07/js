# Tokens de Acceso

Gestiona el acceso API a tu instancia de Engine. Los tokens de acceso son válidos hasta que se eliminen.

### Crear un token de acceso

Concede acceso API a tu instancia de Engine con tokens de acceso de larga duración.

1. Navega al [panel de control de thirdweb Engine](https://thirdweb.com/team/~/~/engine) y selecciona tu instancia de Engine.
1. Selecciona **Tokens de Acceso**.
1. Haz clic en **Crear Token de Acceso**.

Proporciona el encabezado `Authorization: Bearer <access_token>` al llamar a la API de Engine.

<Callout title="¡Un token de acceso solo se muestra una vez!" variant="warning">

Almacena los tokens de acceso de manera segura y elimínalos si se ven comprometidos.

</Callout>

### Eliminar un token de acceso

1. Navega al [panel de control de thirdweb Engine](https://thirdweb.com/team/~/~/engine) y selecciona tu instancia de Engine.
1. Selecciona **Tokens de Acceso**.
1. Junto a un token de acceso, selecciona **... > Eliminar**.
