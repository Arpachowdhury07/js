# Seguridad

Engine te proporciona funciones de seguridad para configurar y restringir el acceso a operaciones sensibles.

## Lista blanca de IP

Puedes restringir el acceso a tu instancia de Engine configurando una lista blanca de IP. Esto se puede configurar en la sección **Configuración** de tu Engine.
Si no se configura una lista blanca de IP, Engine permitirá todas las solicitudes entrantes.

<Callout variant="info" title="Nota">
	Esto no afecta las llamadas desde el panel de control de Engine a tu instancia de Engine.
</Callout>

## Lista blanca de dominios (CORS)

Puedes restringir el acceso a tu instancia de Engine configurando una lista blanca de dominios. Esto se puede configurar en la sección **Configuración** de los ajustes de Engine.
Se pueden agregar múltiples dominios a la lista blanca, y Engine solo aceptará solicitudes de estos dominios. También se puede agregar un dominio comodín para permitir solicitudes desde cualquier dominio.
