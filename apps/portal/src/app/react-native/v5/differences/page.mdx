# Diferencias con React

El paquete `thirdweb` está diseñado para funcionar con React Native, pero hay algunas diferencias que debes tener en cuenta.

### Carteras disponibles

Todas las carteras son compatibles tanto en la web como en React Native

| Componente | React | React Native |
| --- | --- | --- |
| `inAppWallet` | ✅ | ✅ |
| `smartWallet` | ✅ | ✅ |
| `privateKeyAccount` | ✅ | ✅ |
| `walletConnect` | ✅ | ✅ |
| `coinbaseWallet` | ✅ | ✅ |
| `injectedWallet` | ✅ | ✅ |

### Componentes de UI

Algunos componentes de UI pueden no estar implementados aún en React Native. Sin embargo, puedes construir tu propia UI fácilmente con los hooks y funciones proporcionados.

| Componente | React | React Native |
| --- | --- | --- |
| `ConnectButton` | ✅ |  ✅ |
| `ConnectEmbed` | ✅ |  ✅ |
| `TransactionButton` | ✅ |  ✅ |
| `PayEmbed` | ✅ |  ⚠️ (próximamente) |

¿Interesado en contribuir para sacar estos antes? ¡Haz un PR en el [monorepo JS](https://github.com/thirdweb-dev/js)!
