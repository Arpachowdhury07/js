# Unity SDK

El mejor SDK de Unity para construir juegos multiplataforma en cualquier blockchain EVM.

Conéctate a las billeteras de los usuarios, interactúa con contratos inteligentes, firma mensajes y utiliza estándares comunes como tokens, NFTs, marketplaces; todo con URLs RPC integradas, gateways IPFS, Abstracción de Cuentas y más.

## Instalación

El SDK de Unity se distribuye como un [Paquete de Activos](https://docs.unity3d.com/Manual/AssetPackages.html) lo que te permite ver y editar el código fuente.

También envuelve nuestro SDK de [.NET](https://github.com/thirdweb-dev/dotnet) enfocado en juegos, lo que hace que las actualizaciones de versión sean extremadamente simples con solo un cambio en un archivo DLL.

Puedes descargar la última versión del SDK desde nuestra [página de lanzamientos en GitHub](https://github.com/thirdweb-dev/unity-sdk/releases).
