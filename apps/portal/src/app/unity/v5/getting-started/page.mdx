# Primeros pasos

<Callout variant='info' title="Se requiere una clave de API">

Para comenzar con el SDK de Unity, necesitarás tener una [clave de API de thirdweb](https://thirdweb.com/create-api-key) creada. Asegúrate de permitir un ID de paquete para usar dentro de aplicaciones nativas de Unity.

</Callout>

Para comenzar, necesitarás [descargar e instalar Unity Hub y Unity Editor](https://unity.com/download).

Recomendamos usar la versión 2022 LTS de Unity.

## Pasos para la Integración

<Steps>

<Step title="Descarga del Paquete de Unity" >

Descarga el paquete más reciente del SDK de Unity de thirdweb desde la [página de lanzamientos](https://github.com/thirdweb-dev/unity/releases).

</Step>

<Step title="Importación del Paquete de Unity" >

Arrastra y suelta el paquete en tu proyecto de Unity.

</Step>

<Step title="¡Pruébalo!" >

¡Juega con nuestro ejemplo `Scene_Playground` para probar el SDK!

</Step>

<Step title="¡Construyelo!" >

Consulta las [Instrucciones de Construcción](/unity/v5/build-instructions), elige una plataforma de destino y construye tu escena.

Soportamos todas las plataformas.

</Step>

</Steps>

## Thirdweb Manager

¡Ahora estás listo para usar el SDK en tu propia escena!

Recomendamos agregar un prefab de [ThirdwebManager](/unity/v5/thirdwebmanager) a tu escena.

Navega a `Thirdweb` > `Runtime` > `Unity` > `Prefabs` y arrastra el `ThirdwebManager` a tu escena.

Alternativamente, usa nuestro práctico instalador rápido desde el menú superior `Herramientas > Thirdweb`.
