# MetaMaskWallet

`MetaMaskWallet` es el proveedor de billetera externa más popular que permite a los usuarios conectar su billetera MetaMask a tu juego.

Este método de conexión permite conectarse a la extensión del navegador y solo está disponible para construcciones WebGL.

Utiliza [WalletConnectWallet](/unity/v5/wallets/walletconnect) si estás construyendo para plataformas de runtime independientes o móviles.

### Conectando usando MetaMask

```csharp
var options = new WalletOptions(provider: WalletProvider.MetaMaskWallet, chainId: 1);
var wallet = await ThirdwebManager.Instance.ConnectWallet(options);
```

Solicitará al usuario que conecte su extensión de MetaMask en el navegador a tu juego.
