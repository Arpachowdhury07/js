# SmartWallet

Instancia o actualiza cualquier otra billetera a un `SmartWallet` para habilitar interacciones avanzadas con blockchain, incluyendo transacciones sin gas a través de Account Abstraction (ERC4337 así como ZkSync Native AA).

Account Abstraction es un sistema que convierte cualquier billetera personal en una billetera inteligente, permitiéndote patrocinar el gas para tus usuarios y desbloqueando características avanzadas de permisos que permiten una incorporación de usuarios sin fricciones, transacciones sin gas, automatización y mucho más.

Recomendamos usar Smart Wallets como el tipo principal de billetera para tus usuarios.

### Conectando directamente a un Smart Wallet

```csharp
var smartWalletOptions = new SmartWalletOptions(sponsorGas: true)
var options = new WalletOptions(..., smartWalletOptions: smartWalletOptions);
var wallet = await ThirdwebManager.Instance.ConnectWallet(options);
```

### Actualizando una billetera existente a un Smart Wallet

```csharp
var smartWalletOptions = new SmartWalletOptions(sponsorGas: true)
var smartWallet = await ThirdwebManager.Instance.UpgradeToSmartWallet(
    personalWallet: myExistingIThirdwebWallet, 
    chainId: 1, 
    smartWalletOptions: smartWalletOptions
);
```
