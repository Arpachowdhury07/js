# WalletConnectWallet

`WalletConnectWallet` es un proveedor de billetera que permite a los usuarios conectar cualquiera de más de 400 billeteras externas.

Utiliza el protocolo [WalletConnect](https://explorer.walletconnect.com/).

### Conectando usando WalletConnect

```csharp
var options = new WalletOptions(provider: WalletProvider.WalletConnectWallet, chainId: 1);
var wallet = await ThirdwebManager.Instance.ConnectWallet(options);
```

Instanciará `WalletConnectModal` - un modal WalletConnect listo para usar que se puede utilizar para conectar con más de 400 billeteras.

Asegúrate de haber agregado las cadenas a las que deseas conectarte a tu prefab de [ThirdwebManager](/unity/v5/thirdwebmanager).
