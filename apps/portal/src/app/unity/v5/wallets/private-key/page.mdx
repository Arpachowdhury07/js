# PrivateKeyWallet

`PrivateKeyWallet` es mejor utilizado como una billetera de invitado. Debe ser tratada como efímera, útil para pruebas.

### Generando un PrivateKeyWallet

```csharp
var options = new WalletOptions(provider: WalletProvider.PrivateKeyWallet, chainId: 1);
var wallet = await ThirdwebManager.Instance.ConnectWallet(options);
```
