# Vincular Perfiles

Cuando usas [billeteras integradas](/react/v5/in-app-wallet/get-started) o [billeteras del ecosistema](/react/v5/ecosystem-wallets/get-started), puedes vincular perfiles a tu cuenta. Esto permite a los usuarios iniciar sesión con un método sin fricción como invitado, passkey o correo electrónico, y luego vincular su billetera o perfil social a su cuenta.

Una vez vinculados, los usuarios pueden iniciar sesión en su cuenta con cualquiera de sus perfiles vinculados.

## Uso con componentes UI

Al usar [`ConnectButton`](/references/typescript/v5/ConnectButton) o [`useWalletDetailsModal`](/references/typescript/v5/useWalletDetailsModal), los usuarios pueden vincular un nuevo perfil haciendo clic en el botón "Gestionar Billetera", siempre y cuando estén conectados con una billetera integrada o del ecosistema.

## Uso con tu propia UI

Puedes usar la función [`useLinkProfile`](/references/typescript/v5/useLinkProfile) para vincular un nuevo perfil a tu cuenta. Esto es útil si deseas usar una UI personalizada para el flujo de vinculación.
