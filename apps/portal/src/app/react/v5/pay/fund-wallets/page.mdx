## Uso con Componentes UI

La forma más fácil de financiar billeteras es usar el componente UI preconstruido `PayEmbed`. Ten en cuenta que el `ConnectButton` también tiene un flujo de financiamiento integrado que también puede ser personalizado.

```tsx
import { PayEmbed } from "thirdweb/react";

function App() {
  return (
    <PayEmbed
      client={client}
      payOptions={{
        mode: "fund_wallet",
        metadata: {
          name: "Get funds",
        },
        prefillBuy: {
          chain: base,
          amount: "0.01",
        },
        // ... theme, currency, amounts, payment methods, etc.
      }}
    />
  );
}
```

Consulta la [referencia de la API de PayEmbed](/references/typescript/v5/PayEmbed) para más información.
