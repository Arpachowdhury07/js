# React SDK

Además de la API principal de TypeScript, `thirdweb/react` exporta varios componentes y hooks para conectar carteras e interactuar fácilmente con contratos inteligentes.

## Instalación

<InstallTabs
	npm="npm i thirdweb"
	yarn="yarn add thirdweb"
	pnpm="pnpm i thirdweb"
	bun="bun i thirdweb"
/>

## Kits de inicio

Puedes crear rápidamente un nuevo proyecto con la CLI de thirdweb

```bash
npx thirdweb create app
```

o clonar el repositorio de inicio de Next.js o Vite:
