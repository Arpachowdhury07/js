# Componentes de UI

Puedes usar el componente [`ConnectButton`](/react/v5/ConnectButton) o [`ConnectEmbed`](/react/v5/ConnectEmbed) para una interfaz rápida, fácil y personalizable.

Estos componentes proporcionan una interfaz preconstruida para conectar varias carteras y se encargan de muchos casos específicos de cada cartera, para que puedas concentrarte en construir tu aplicación.

```tsx
import { createThirdwebClient } from "thirdweb";
import { ThirdwebProvider, ConnectButton } from "thirdweb/react";

const client = createThirdwebClient({ clientId });
 
export default function App() {
  return (
    <ThirdwebProvider>
      <ConnectButton client={client} />
    </ThirdwebProvider>
  );
}
```

Estos componentes son compatibles con más de 500 carteras, incluyendo carteras en la aplicación y abstracción de cuentas.

También muestra automáticamente todas las extensiones de carteras compatibles con `EIP-6963` instaladas por el usuario.

## Crear Carteras

Puedes crear cualquier cartera por id con autocompletado usando la función [`createWallet`](/references/typescript/v5/createWallet). O usa una de las carteras de primera parte como [`inAppWallet`](/references/typescript/v5/inAppWallet) o [`ecosystemWallet`](/references/typescript/v5/ecosystemWallet).

## Después de la Conexión

Una vez que la cartera esté conectada, puedes usar los [hooks de conexión](/react/v5/connecting-wallets/hooks) para obtener información sobre la cartera conectada, como la dirección, la cuenta, etc.
