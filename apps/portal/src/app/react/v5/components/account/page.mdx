# Account UI Components

Construye tu propia interfaz para mostrar la información de la cuenta con estos componentes sin cabeza.

```tsx
import {
  AccountProvider,
  AccountAvatar,
  AccountName,
  AccountBalance,
} from "thirdweb/react";

const App = () => (
<AccountProvider>
  <AccountAvatar />
  <AccountName />
  <AccountBalance />
</AccountProvider>
);
```
