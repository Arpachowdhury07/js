# Conectar usuarios a tu Ecosistema

Las carteras del ecosistema heredan toda la funcionalidad de [carteras dentro de la aplicación](/react/v5/in-app-wallet/get-started), pero en lugar de estar limitadas a una sola aplicación, pueden usarse en múltiples aplicaciones.

Dirígete al [panel de control de thirdweb](https://thirdweb.com/team) para crear un ecosistema y obtener tu id de ecosistema.

## Configurar carteras del ecosistema

La única diferencia con las carteras dentro de la aplicación es cómo creas una cartera del ecosistema usando la función `ecosystemWallet()` y pasando tu id de ecosistema.

```tsx
import { ecosystemWallet } from "thirdweb/wallets";

const wallet = ecosystemWallet("ecosystem.your-ecosystem-id");
```

## Configuración de estrategias de autenticación

Puedes configurar las estrategias de autenticación permitidas en [tu panel de control](https://thirdweb.com/team/~/~/ecosystem).

<DocImage src={EcosystemSocialConfig}/>

Para las interfaces de usuario preconstruidas, las estrategias de autenticación habilitadas se leen automáticamente desde las configuraciones de tu panel de control.

Para interfaces de usuario personalizadas, puedes configurar las opciones de autenticación al conectar la cartera. Consulta [usando tu propia interfaz de usuario](#usando-tu-propia-ui) para un ejemplo.

<Callout variant="warning" title="Uso de estrategias de autenticación deshabilitadas">
	Nota que, aunque hoy se permite pasar cualquier estrategia de autenticación, recomendamos pasar solo las estrategias habilitadas en tu panel de control. Podríamos hacer cumplir esto en el futuro.
</Callout>

## Pasando un ID de socio

Para ecosistemas cerrados, puedes invitar a socios a tu ecosistema. Los socios deberán pasar un `partnerId` válido a la función `ecosystemWallet()` para poder conectarse a tu ecosistema.

```tsx
const wallet = ecosystemWallet("ecosystem.your-ecosystem-id", {
	partnerId: "your-partner-id",
});
```

Para más información, consulta la [referencia de la API de ecosystemWallet](/references/typescript/v5/ecosystemWallet).
