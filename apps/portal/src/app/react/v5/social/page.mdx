Consulta todas las identidades sociales web3 para una dirección dada.

## Uso con componentes UI

Al usar [`ConnectButton`](/references/typescript/v5/ConnectButton) o [`useWalletDetailsModal`](/references/typescript/v5/useWalletDetailsModal), el avatar y el nombre de usuario del usuario se poblarán automáticamente con sus perfiles sociales vinculados.

## Uso con tu propia UI

Usa el hook [`useSocialProfiles`](/references/typescript/v5/useSocialProfiles) para consultar todas las identidades sociales web3 para una dirección dada.
