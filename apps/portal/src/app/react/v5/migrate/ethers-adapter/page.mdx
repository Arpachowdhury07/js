# Ethers.js en v5

Una idea errónea común sobre el SDK de thirdweb v5 es que ya no funcionará con Ethers.js.  
De hecho, el SDK v5 está diseñado para funcionar bien con SDKs web3 populares como viem, ethers v5 y ethers v6.

Supongamos que decides actualizar a v5 y aún quieres seguir usando ethers.js,  
¡puedes utilizar el adaptador de ethers de thirdweb para lograrlo!

### Signer

En v5, en lugar de tener "signer", pasamos a [`Account`](/references/typescript/v5/Account), que es el reemplazo de "signer".  
Gracias al adaptador de ethers de thirdweb, puedes convertir de manera conveniente entre el Signer de ethers y el Account de thirdweb.

```ts
import { ethers5Adapter } from "thirdweb/adapters/ethers5";
import { useActiveAccount } from "thirdweb/react";

const account = useActiveAccount();
...
const signer = await ethers5Adapter.signer.toEthers({ client, chain, account });
```
Una vez que tengas el signer, puedes realizar diferentes tareas usando ethers.js como de costumbre:

```tsx
// Get balance
const balance = await signer.getBalance();

// Sign message
const signature = await signer.signMessage(message);
```

### Provider
Similar al `signer`, puedes obtener el proveedor de ethers en thirdweb v5 utilizando el mismo adaptador:

```ts
import { ethers5Adapter } from "thirdweb/adapters/ethers5";
import { createThirdwebClient } from "thirdweb";

const client = createThirdwebClient({
  clientId: "...",
});

const provider = ethers5Adapter.provider.toEthers({ client, chainId });
```

y realizar tareas con él:

```ts
const blockNumber = await provider.getBlockNumber();
```

### Contratos
De manera similar a los 2 componentes anteriores, así es como puedes convertir una [instancia de contrato de thirdweb](/references/typescript/v5/ThirdwebContract) a ethers:
```tsx
import { ethers5Adapter } from "thirdweb/adapters/ethers5";
const ethersContract = await ethers5Adapter.contract.toEthers({
  thirdwebContract,
});
```
and then do stuff with it:
```tsx
// Estimate gas
const gasLimit = await contract.estimateGas["functionName"](...params);

// Send a transaction
const tx = await contract["functionName"](...params, { gasLimit });
```
