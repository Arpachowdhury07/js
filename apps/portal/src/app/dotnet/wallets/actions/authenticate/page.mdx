# IThirdwebWallet.Authenticate

Inicia sesión en tu backend integrado con [thirdweb Auth](https://portal.thirdweb.com/connect/auth).

## Uso

```csharp
var authResult = await wallet.Authenticate<string>(domain: "https://mydomain.com", chainId: 1);
```

<Details summary="Parámetros">

### wallet

La instancia de la wallet que firmará el payload SIWE (Sign-In With Ethereum), debe ser un `IThirdwebWallet`.

### domain

El dominio del backend con el que autenticarse, debe ser un `string`.

### chainId

El ID de la cadena del blockchain con el que autenticarse, debe ser un `BigInteger`.

### authPayloadPath (opcional)

La ruta al endpoint del payload de autenticación, por defecto es `"/auth/payload"`, debe ser un `string`.

### authLoginPath (opcional)

La ruta al endpoint de inicio de sesión de autenticación, por defecto es `"/auth/login"`, debe ser un `string`.

### authPayloadMethod (opcional)

El método HTTP para la solicitud del payload de autenticación, puede ser `"GET"` o `"POST"`. Por defecto es `"GET"`.

### authLoginMethod (opcional)

El método HTTP para la solicitud de inicio de sesión de autenticación, puede ser `"GET"` o `"POST"`. Por defecto es `"POST"`.

### separatePayloadAndSignatureInBody (opcional)

Si separar el payload y la firma en el cuerpo de la solicitud de inicio de sesión, debe ser un `booleano`. Por defecto es `false`.

### httpClientOverride (opcional)

Una instancia de `IThirdwebHttpClient` para sobrescribir el cliente HTTP por defecto, por defecto es `null`.

</Details>

<Details summary="Valor de retorno">

### T

El resultado de la autenticación como el tipo `T`. Puede ser `string`, `byte[]` o un objeto deserializado.

</Details>

## Ejemplo

```csharp
// Ejemplo de uso retornando el resultado de autenticación como un string
var authResult = await wallet.Authenticate<string>(domain: "https://mydomain.com", chainId: 1);

// Ejemplo de uso retornando el resultado de autenticación como un array de bytes
var authResultBytes = await wallet.Authenticate<byte[]>(domain: "https://mydomain.com", chainId: 1);

// Ejemplo de uso retornando el resultado de autenticación como un objeto personalizado
var authResultObject = await wallet.Authenticate<MyCustomType>(domain: "https://mydomain.com", chainId: 1);
```
