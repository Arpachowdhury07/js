# SmartWallet.RemoveAdmin

Elimina un administrador de la Smart Wallet conectada.

## Uso


```csharp
var receipt = await SmartWallet.RemoveAdmin("0xLivedminAddress");
```

<Details summary="Parámetros">

### admin

Cadena hexadecimal `string` que representa el firmante al que se le eliminará el acceso de administrador.

</Details>

<Details summary="Valor de retorno">

### receipt

El recibo de la transacción como un objeto `TransactionReceipt`.

</Details>
