# EcosystemWallet.GenerateExternalLoginLink

Este método devuelve una URL que se puede usar para transferir la sesión iniciada de In-App o Ecosystem Wallets de .NET a aplicaciones React construidas con thirdweb.

Configura tu sitio web de React para usar thirdweb Connect, abre el enlace generado por este método y tus usuarios se conectarán automáticamente.
Si estás usando Ecosystem Wallets, asegúrate de crear la billetera del ecosistema con el mismo ID de ecosistema y ID de socio (si aplica) e incluirla como una de tus billeteras en, por ejemplo, la propiedad `wallets` de tu `ConnectButton`.
Asegúrate de usar la misma clave API en ambos lados.

## Uso


```csharp
var url = await ecosystemWallet.GenerateExternalLoginLink();
```
