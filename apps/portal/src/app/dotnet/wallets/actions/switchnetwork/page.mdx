# IThirdwebWallet.SwitchNetwork

Este método permite establecer la cadena activa de la billetera si es aplicable.

Cuando uses Smart Wallets, asegúrate de que cualquier sobrecarga a los contratos predeterminados, como fábricas, esté desplegada en todas las cadenas a las que planeas cambiar. También soportamos el cambio entre cadenas zksync y no zksync.

## Uso


```csharp
await wallet.SwitchNetwork(chainId);
```

<Details summary="Parámetros">

### chainId (requerido)

El ID de la cadena a la cual deseas cambiar la billetera.

</Details>
