# EcosystemWallet.GetUserDetails

Este método devuelve información sobre la billetera de usuario conectada y sus cuentas vinculadas.

## Uso


```csharp
var result = await ecosystemWallet.GetUserDetails();
```

<Details summary="Valor de retorno">

### UserStatusResponse

La información del usuario y las cuentas vinculadas, como la dirección del usuario, correo electrónico y más.

</Details>
