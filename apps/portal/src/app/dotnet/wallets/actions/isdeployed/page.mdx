# SmartWallet.IsDeployed

Este método obtiene si la Smart Wallet conectada está desplegada en la cadena.

## Uso


```csharp
bool isDeployed = await SmartWallet.IsDeployed();
```

<Details summary="Return Value">

### bool

Indica si una Smart Wallet está desplegada como un `bool`.
