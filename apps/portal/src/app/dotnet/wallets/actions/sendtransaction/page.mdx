# IThirdwebWallet.SendTransaction

Envía una transacción sin procesar desde la billetera conectada (si es compatible) y devuelve el hash. Se recomienda usar `ThirdwebContract.Write` en su lugar.

## Uso

```csharp
string transactionHash = await wallet.SendTransaction(new ThirdwebTransactionInput(...));
```

<Details summary="Parámetros">

### ThirdwebTransactionInput

Objeto `ThirdwebTransactionInput` que contiene los detalles necesarios de la transacción, como `To`, `Data`, `Value`.

</Details>

<Details summary="Valor de retorno">

### string

El hash de la transacción como un `string`.

</Details>
