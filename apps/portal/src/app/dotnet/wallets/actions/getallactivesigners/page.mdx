# SmartWallet.GetAllActiveSigners

Este método recupera una lista de todos los firmantes activos con sus permisos para una smart wallet.

## Uso


```csharp
// Get all active signers
List<SignerPermissions> signers = await smartWallet.GetAllActiveSigners();
```

<Details summary="Parámetros">

### Ninguno

Este método no requiere parámetros.

</Details>

<Details summary="Valor de retorno">

### Lista de `SignerPermissions`

El método devuelve una lista de objetos `SignerPermissions`, cada uno representando un firmante y sus permisos.

Cada objeto `SignerPermissions` contiene las siguientes propiedades:

- `Signer`: La dirección del firmante.
- `ApprovedTargets`: Una lista de direcciones con las que el firmante está aprobado para interactuar.
- `NativeTokenLimitPerTransaction`: La cantidad máxima de tokens nativos que el firmante puede transferir por transacción.
- `StartTimestamp`: El tiempo de inicio de los permisos del firmante.
- `EndTimestamp`: El tiempo de finalización de los permisos del firmante.

</Details>
