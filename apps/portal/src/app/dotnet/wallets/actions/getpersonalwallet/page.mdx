# GetPersonalWallet

Este método recupera el firmante actual del SmartWallet conectado.

## Uso


```csharp
var signer = await smartWallet.GetPersonalWallet();
```

<Details summary="Valor de retorno">

### IThirdwebWallet

El método devuelve una instancia de `IThirdwebWallet` que representa el firmante actual del SmartWallet conectado.

</Details>
