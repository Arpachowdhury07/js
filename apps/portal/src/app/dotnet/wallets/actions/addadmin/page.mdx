# SmartWallet.AddAmin

Agrega un nuevo administrador a la billetera inteligente conectada.

## Uso


```csharp
var receipt = await SmartWallet.AddAdmin("0xNewAdminAddress");
```

<Details summary="Parámetros">

### admin

Cadena hexadecimal `string` que representa el firmante al que se le otorgará acceso de administrador.

</Details>

<Details summary="Valor de retorno">

### receipt

El recibo de la transacción como un objeto `TransactionReceipt`.

</Details>
