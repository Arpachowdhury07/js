# EcosystemWallet.GetUserDetails

Este método devuelve información sobre los detalles del proveedor de autenticación del usuario conectado, por ejemplo, detalles específicos de Google o Github.

## Uso


```csharp
var result = ecosystemWallet.GetUserAuthDetails();
```

<Details summary="Valor de Retorno">

### JObject

Los detalles del proveedor de autenticación del usuario como un `JObject`.
