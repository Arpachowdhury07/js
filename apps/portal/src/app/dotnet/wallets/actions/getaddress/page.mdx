# IThirdwebWallet.GetAddress

Este método recupera la dirección de la blockchain asociada con una instancia de billetera Thirdweb, lo que permite a los desarrolladores identificar e interactuar fácilmente con la billetera en la blockchain.

## Uso

```csharp
string walletAddress = await wallet.GetAddress();
```

<Details summary="Valor de retorno">

`string`

La dirección de blockchain como una `string` hexadecimal. Esta dirección identifica de manera única la billetera en la blockchain, permitiendo interacciones como transacciones o llamadas a contratos inteligentes.

</Details>
