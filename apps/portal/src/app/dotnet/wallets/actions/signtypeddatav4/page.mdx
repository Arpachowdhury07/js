# IThirdwebWallet.SignTypedDataV4

Firma datos estructurados de acuerdo con EIP-712, mejorando la seguridad y la usabilidad de la firma de datos estructurados al proporcionar una experiencia de usuario estandarizada en las aplicaciones de Ethereum.

## Uso

```csharp
string signature = await wallet.SignTypedDataV4(jsonTypedData);
```

<Details summary="Parámetros">

### jsonTypedData (requerido)

Una cadena JSON que representa los datos tipados a ser firmados. Estos datos deben seguir el estándar EIP-712, incluyendo tipos, dominio y mensaje.

</Details>

<Details summary="Valor de retorno">

### string

Devuelve un `string` que representa la firma hexadecimal de los datos tipados. Esta firma puede ser verificada para confirmar la integridad de los datos y la autenticidad del firmante.

</Details>
