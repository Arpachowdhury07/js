# IThirdwebWallet.PersonalSign

Este método permite firmar un mensaje de cadena utilizando la clave privada de la billetera. El mensaje firmado demuestra la integridad y autenticidad del mensaje, demostrando que el mensaje fue creado por el propietario de la clave privada.

**Nota:** Al utilizar `SmartWallet`, al llamar a este método desde una cuenta inteligente no desplegada se hará uso de [EIP-6492](https://eips.ethereum.org/EIPS/eip-6492).

## Uso


```csharp
string signature = await wallet.PersonalSign("Your message here");
```

<Details summary="Parámetros">

### message (requerido)

Un mensaje de tipo `string` que debe ser firmado. Este es el mensaje cuya integridad y autenticidad deseas asegurar.

</Details>

<Details summary="Valor de retorno">

### string

El método devuelve un `string` que representa la firma hexadecimal del mensaje. Esta firma puede ser utilizada para verificar la autenticidad del mensaje.

</Details>
