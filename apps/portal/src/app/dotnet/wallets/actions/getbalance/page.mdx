# IThirdwebWallet.GetBalance

Este método permite obtener el saldo de tokens nativos o el saldo de un token específico para una billetera.

## Uso


```csharp
// Get native balance
BigInteger nativeBalance = await wallet.GetBalance(chainId);

// Get ERC20 token balance
BigInteger tokenBalance = await wallet.GetBalance(chainId, erc20ContractAddress);
```

<Details summary="Parámetros">

### wallet (obligatorio)

Una instancia de `IThirdwebWallet`. Esto representa la billetera para la cual deseas obtener el saldo.

### chainId (obligatorio)

El ID de la cadena desde la cual deseas obtener el saldo.

### erc20ContractAddress (opcional)

La dirección del contrato ERC20 opcional desde la cual deseas obtener el saldo. Si no se proporciona, el método devolverá el saldo del token nativo.

</Details>

<Details summary="Valor de retorno">

### string

El método devuelve un `BigInteger` representando el saldo en Wei.

</Details>
