# IThirdwebWallet.Transfer

Este método transfiere una cantidad especificada de Wei (la denominación más pequeña de Ether) o tokens ERC20 a una dirección de billetera especificada en una cadena dada.

## Uso


```csharp
// Transfer Wei
var receipt = await wallet.Transfer(chainId, toAddress, weiAmount);
// Transfer ERC20
var receipt = await wallet.Transfer(chainId, toAddress, weiAmount, erc20ContractAddress);
```

<Details summary="Parámetros">

### wallet (requerido)

Una instancia de `IThirdwebWallet`. Representa la billetera desde la cual se está transfiriendo el Wei.

### chainId (requerido)

El ID de la cadena de la red en la que se realizará la transferencia.

### toAddress (requerido)

La dirección a la cual se transferirá el Wei.

### weiAmount (requerido)

La cantidad de Wei a transferir. Debe ser 0 o mayor.

### erc20ContractAddress (opcional)

La dirección del contrato del token ERC20 a transferir. Si no se proporciona, la transferencia será en tokens nativos.

</Details>

<Details summary="Valor de retorno">

### ThirdwebTransactionReceipt

El método devuelve un `ThirdwebTransactionReceipt` que contiene detalles sobre la transacción.

</Details>

## Excepciones

- `ArgumentNullException`: Lanzado cuando `wallet` es nulo.
- `ArgumentOutOfRangeException`: Lanzado cuando `chainId` es menor o igual a 0.
- `ArgumentException`: Lanzado cuando la dirección del destinatario `toAddress` es nula o vacía.
- `ArgumentOutOfRangeException`: Lanzado cuando `weiAmount` es menor a 0.
