# SmartWallet.GetAllAdmins

Este método recupera una lista de todas las direcciones de administradores de un Smart Wallet.

## Uso


```csharp
// Get all admin addresses
List<string> admins = await smartWallet.GetAllAdmins();
```

<Details summary="Parámetros">

### Ninguno

Este método no requiere parámetros.

</Details>

<Details summary="Valor de retorno">

### Lista de `string`

El método devuelve una lista de objetos `string`, cada uno representando una dirección de administrador.

</Details>
