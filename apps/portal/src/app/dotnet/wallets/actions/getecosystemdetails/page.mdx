# EcosystemWallet.GetEcosystemDetails

Este método devuelve información sobre el Ecosistema al que pertenece la billetera.

## Uso


```csharp
var result = await ecosystemWallet.GetEcosystemDetails();
```

<Details summary="Valor de retorno">

### EcosystemDetails

Los detalles del Ecosistema, como el nombre del Ecosistema, el slug, la URL de la imagen y más.

</Details>
