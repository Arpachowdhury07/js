# IThirdwebWallet.Disconnect

Este método desconecta limpiamente la instancia de la billetera de la red blockchain o de cualquier servicio relacionado, finalizando efectivamente la sesión.

**Nota:** Al usar `InAppWallet`, esto borrará la sesión y requerirá que el usuario vuelva a autenticarse para reconectarse más tarde. Al usar `SmartWallet`, esto desconectará la billetera personal que utiliza como firmante.

## Uso

```csharp
await wallet.Disconnect();
```

<Details summary="Descripción">

El método `Disconnect` se usa para terminar cualquier conexión o sesión activa que la billetera pueda tener con la red blockchain o servicios relacionados. Esto puede incluir desconectar de un proveedor web3, borrar cualquier autenticación almacenada en caché u otras acciones de limpieza necesarias para asegurar un estado limpio.

</Details>
