# IThirdwebWallet.GetTransactionCount

Este método recupera el contador de transacciones (es decir, nonce) para una dirección de billetera especificada en una cadena dada.

## Uso


```csharp
BigInteger transactionCount = await wallet.GetTransactionCount(chainId, address);
```

<Details summary="Parámetros">

### wallet (obligatorio)

Una instancia de `IThirdwebWallet`. Esto representa la billetera para la cual deseas obtener el contador de transacciones.

### chainId (obligatorio)

El ID de la cadena desde la cual deseas obtener el contador de transacciones.

### address (obligatorio)

La dirección de la billetera para la cual deseas obtener el contador de transacciones.

### blocktag (opcional)

La etiqueta del bloque para obtener el contador de transacciones. Por defecto es `"pending"`. Puedes configurarlo como `"latest"` o `"earliest"`.

</Details>

<Details summary="Valor de retorno">

### BigInteger

El método devuelve un `BigInteger` representando el contador de transacciones (nonce) de la billetera.

</Details>
