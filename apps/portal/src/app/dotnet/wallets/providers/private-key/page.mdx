# PrivateKeyWallet.Create

Instancia un `PrivateKeyWallet` con una clave privada dada. Esta billetera es capaz de firmar transacciones y mensajes, interactuar con contratos inteligentes y realizar otras operaciones en la blockchain. **Este tipo de billetera está destinada solo para aplicaciones backend**, debido a la naturaleza sensible de las claves privadas.

## Uso


```csharp
var wallet = await PrivateKeyWallet.Create(client, "yourPrivateKeyHex");
```

<Callout variant="warning" title="Importante">
    Este método implica el uso de una clave privada sin procesar, que debe manejarse con la máxima precaución. Nunca expongas claves privadas en el código del cliente ni las almacenes de manera insegura. Idealmente, usa variables de entorno o bóvedas seguras para gestionar las claves privadas en servicios backend.
</Callout>

<Details summary="Parámetros">

### client (requerido)

Una instancia de `ThirdwebClient`.

### privateKeyHex (requerido)

La clave privada de la billetera, proporcionada como un `string` hexadecimal. Esta clave otorga control total sobre los fondos y activos de la billetera en la blockchain.

</Details>

<Details summary="Valor de retorno">
PrivateKeyWallet

Devuelve una instancia de `PrivateKeyWallet`, lista para firmar transacciones y mensajes con la clave privada proporcionada.

</Details>

## Ejemplo

Aquí tienes un ejemplo de cómo crear un PrivateKeyWallet para firmar un mensaje simple. Este ejemplo asume que ya tienes una instancia de ThirdwebClient creada:

```csharp
var client = ThirdwebClient.Create("yourClientId");
var privateKeyHex = "yourPrivateKeyHex"; // Should be securely stored and accessed
var wallet = await PrivateKeyWallet.Create(client, privateKeyHex);

Console.WriteLine($"PrivateKeyWallet address: {await wallet.GetAddress()}");

// Sign a message
var message = "Hello, Thirdweb!";
var signature = await wallet.PersonalSign(message);
Console.WriteLine($"Signature: {signature}");
```

**Nota**: El `PrivateKeyWallet` está destinado a ser utilizado en entornos seguros y backend. Asegúrate de que las claves privadas nunca sean expuestas ni codificadas de manera fija en aplicaciones del lado del cliente. Usa `InAppWallet` y `SmartWallet` para aplicaciones del lado del cliente.
