# SmartWallet.Create

Instancia un `SmartWallet` para habilitar interacciones avanzadas con blockchain, incluyendo transacciones sin gas a través de la abstracción de cuentas. Este tipo de billetera es especialmente útil para crear una experiencia amigable para el usuario en aplicaciones descentralizadas.

## Uso


```csharp
var smartWallet = await SmartWallet.Create(personalWallet, chainId);
```

<Callout variant="info" title="Funcionalidad Avanzada">
	Este método configura un SmartWallet con transacciones opcionales sin gas, aprovechando la abstracción de cuentas. Está diseñado para desarrolladores que buscan crear interacciones blockchain sin interrupciones dentro de sus aplicaciones.
</Callout>

<Details summary="Parámetros">

### personalWallet (requerido)

Una instancia de IThirdwebWallet (por ejemplo, `PrivateKeyWallet` o `InAppWallet`). Representa la billetera personal conectada al SmartWallet.

### chainId (requerido)

El ID de la cadena de la red blockchain como un `BigInteger`. Identifica la red con la que interactúa el SmartWallet.

### factoryAddress (opcional)

La dirección del contrato de fábrica utilizado para desplegar nuevas instancias de SmartWallet.

### gasless (opcional)

Un `booleano` que indica si la billetera debe operar en modo sin gas para las transacciones.

### accountAddressOverride (opcional)

Sobrescribe la dirección de cuenta predeterminada para el SmartWallet. Útil para casos de uso avanzados de claves de sesión.

### entryPoint (opcional)

La dirección del contrato de entryPoint para la abstracción de cuentas.

### bundlerUrl (opcional)

La URL del servidor de bundler utilizado para procesar las Operaciones de Usuario.

### paymasterUrl (opcional)

La URL del servicio de paymaster que financia el gas para las Operaciones de Usuario.

### tokenPaymaster (opcional)

El `TokenPaymaster` (enum) que representa el token ERC20 con el que quieres que los usuarios paguen las tarifas de gas.

</Details>

<Details summary="Valor de retorno">

### SmartWallet

Devuelve una instancia de `SmartWallet`, configurada con los parámetros proporcionados y lista para las interacciones blockchain.

</Details>
Ejemplo

Así es como puedes crear un SmartWallet para un usuario, suponiendo que ya tienes configurado un ThirdwebClient y una billetera personal (PrivateKeyWallet o InAppWallet):

```csharp
var client = ThirdwebClient.Create(clientId: "yourClientId", bundleId: "yourBundleId");
var personalWallet = await PrivateKeyWallet.Create(client, "yourPrivateKeyHex");

BigInteger chainId = 137; // Polygon mainnet
bool gasless = true; // Enable gasless transactions

var smartWallet = await SmartWallet.Create(
    personalWallet,
    gasless,
);

Console.WriteLine($"SmartWallet address: {await smartWallet.GetAddress()}");

// Sign a message (this will also deploy your smart wallet if it hasn't been deployed yet)
var message = "Hello, Thirdweb!";
var signature = await wallet.PersonalSign(message);
Console.WriteLine($"Signature: {signature}");
```
