# ThirdwebTransaction.Create

<Callout variant="warning" title="Caso de uso avanzado">
	Si estás interactuando con un contrato, es mejor usar
	[ThirdwebContract.Prepare](/dotnet/contracts/prepare) para crear una transacción.
</Callout>

El método `ThirdwebTransaction.Create` permite a los usuarios avanzados crear manualmente transacciones usando credenciales específicas de la billetera y entradas de transacción. Este método es ideal para usuarios que necesitan control preciso sobre los detalles de la transacción.

## Uso

```csharp
ThirdwebTransaction transaction = await ThirdwebTransaction.Create(wallet, txInput, chainId);
```

<Details summary="Parámetros">

### wallet (requerido)

Una instancia de `IThirdwebWallet`. Representa la billetera que se utilizará para realizar la transacción. La dirección de la billetera debe coincidir con la dirección desde la cual se especifica en txInput, si se proporciona.

### txInput (requerido)

Una instancia de `ThirdwebTransactionInput`. Esta estructura incluye los detalles necesarios, como la dirección de origen, el valor, el límite de gas y los datos de la transacción.

El campo `from` es opcional y, si no se proporciona, se establecerá por defecto en la dirección de la billetera.

### chainId (requerido)

El ID de cadena de la red blockchain como un `BigInteger`. Este ID especifica a qué red está destinada la transacción.

</Details>

<Details summary="Valor de retorno">

### ThirdwebTransaction

Devuelve una instancia de `ThirdwebTransaction` configurada con los parámetros proporcionados, lista para interactuar con la blockchain.

</Details>
