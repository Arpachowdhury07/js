# Métodos de Instancia de ThirdwebTransaction

La clase `ThirdwebTransaction` proporciona un conjunto de métodos de instancia para establecer varias propiedades de los datos de entrada y la configuración de gas de tu transacción.

Ten en cuenta que en la mayoría de los casos, estos se autocompletan automáticamente por el SDK y no es necesario configurarlos manualmente.

## ThirdwebTransaction.SetTo

Establece la dirección objetivo o el contrato de la transacción.

```csharp
ThirdwebTransaction transaction = transaction.SetTo("0xToAddress");
```

<Details summary="Parámetros">

### toAddress (requerido)

La dirección objetivo o contrato de la transacción. Debe ser una cadena hexadecimal `string`.

</Details>

<Details summary="Valor de Retorno">

### ThirdwebTransaction

Devuelve la instancia de `ThirdwebTransaction` con la propiedad `to` configurada.

</Details>

## ThirdwebTransaction.SetZkSyncOptions

Establece las opciones de ZkSync de la transacción.

Esto te permite usar la Abstracción de Cuentas nativas de ZkSync con cualquier paymaster.

```csharp
ZkSyncOptions zkSyncOptions = new ZkSyncOptions("0xPaymaster", "0xPaymasterInput");
ThirdwebTransaction transaction = transaction.SetZkSyncOptions(zkSyncOptions);
```

<Details summary="Parámetros">

### zkSyncOptions (requerido)

Las opciones de ZkSync de la transacción. Debe ser una instancia de `ZkSyncOptions`.

`ZkSyncOptions` incluye las siguientes propiedades:

- `Paymaster` - La dirección del paymaster. Debe ser una cadena hexadecimal `string`.
- `PaymasterInput` - Los datos de entrada del paymaster. Debe ser una cadena hexadecimal `string`.
- `GasPerPubdataByteLimit` - El límite de gas por byte de pubdata. Debe ser un `BigInteger`.
- `FactoryDeps` - Una lista de dependencias de fábrica. Debe ser una lista de `byte[]`.

</Details>

<Details summary="Valor de Retorno">

### ThirdwebTransaction

Devuelve la instancia de `ThirdwebTransaction` con la propiedad `zkSyncOptions` configurada.

</Details>
