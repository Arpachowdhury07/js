# Utilidades

## Constants.ADDRESS_ZERO

A `string` representing the zero address `0x0000000000000000000000000000000000000000`.

Útil para pasar la dirección cero a varias funciones.

## Constants.NATIVE_TOKEN_ADDRESS

Una `cadena` que representa la dirección del token nativo `0xEeeeeEeeeEeEeeEeEeEeeEEEeeeeEeeeeeeeEEeE`.

Útil para pasar la dirección del token nativo (ETH o equivalente) a varias funciones.

## Constants.DECIMALS_18

Un `doble` que representa el valor `1000000000000000000`.

Útil para convertir valores entre diferentes lugares decimales.

## Constants.ENTRYPOINT_ADDRESS_V06

Una `cadena` que representa la dirección del entrypoint para Account Abstraction 0.7.0 `0x5FF137D4b0FDCD49DcA30c7CF57E578a026d2789`.

## Constants.ENTRYPOINT_ADDRESS_V07

Una `cadena` que representa la dirección del entrypoint para Account Abstraction 0.7.0 `0x0000000071727De22E5E9d8BAf0edAc6f37da032`.

## GetSocialProfiles

Obtiene los perfiles sociales para la dirección o ENS dada usando un cliente Thirdweb.

```csharp

```

<Details summary="Parámetros">

### client

Un objeto `ThirdwebClient` que representa el cliente que se utilizará para obtener los perfiles sociales.

### addressOrEns

Una `cadena` que representa la dirección de la billetera o ENS para obtener los perfiles sociales.

</Details>

<Details summary="Valor de retorno">

### socialProfiles

Un objeto `SocialProfiles` que contiene los perfiles sociales obtenidos.

</Details>
