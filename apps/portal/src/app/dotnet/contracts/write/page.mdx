# ThirdwebContract.Write

El método `ThirdwebContract.Write` te permite ejecutar transacciones que alteran el estado de la blockchain a través de un contrato inteligente. Esto se utiliza típicamente para funciones que crean, modifican o eliminan datos en la blockchain.

## Uso

```csharp
var transactionReceipt = await contract.Write(wallet, contract, "methodName", weiValue, parameters);
```

<Details summary="Parámetros">

### wallet (requerido)

Una instancia de `IThirdwebWallet`. Representa al firmante de la transacción, que puede ser cualquier tipo de proveedor de billetera.

### contract (requerido)

Una instancia de `ThirdwebContract`. Representa el contrato inteligente con el que deseas interactuar.

### methodName (requerido)

El nombre del método del contrato inteligente que deseas llamar. Debe ser un `string`.

### weiValue (requerido)

La cantidad de Wei para enviar con las transacciones pagaderas como `BigInteger`. A menudo se establece en 0 para métodos que no requieren enviar Ether.

### parameters (opcional)

Los parámetros a pasar al método del contrato inteligente, si los hay. Proporciónalos como una lista separada por comas después de weiValue.

</Details>

<Details summary="Valor de retorno">

### TransactionReceipt

Devuelve el recibo de la transacción como un objeto `TransactionReceipt`, que contiene información sobre la transacción ejecutada.

</Details>

## Ejemplo

Supongamos que tienes un contrato inteligente con un método `transfer` que transfiere tokens ERC20 desde la dirección del llamador a otra dirección. Aquí te mostramos cómo podrías ejecutar una transacción para transferir tokens utilizando `ThirdwebContract.Write`:

```csharp
BigInteger chainId = 1; // Ethereum mainnet
string contractAddress = "0x..."; // Your contract address
var client = ThirdwebClient.Create(secretKey: "yourSecretKey");
var contract = await ThirdwebContract.Create(client, contractAddress, chainId);

// The wallet that signs and sends the transaction
var wallet = await PrivateKeyWallet.Create(client, "yourPrivateKeyHex");

// Assuming transfer takes an address and an amount as parameters
string toAddress = "0x...";
BigInteger amount = new BigInteger(1000); // The amount to transfer

// No ether is being sent in this non-payable transaction, so weiValue is 0
BigInteger weiValue = BigInteger.Zero;

// Executing the transfer
var receipt = await contract.Write(wallet, contract, "transfer", weiValue, toAddress, amount);
Console.WriteLine($"Transaction receipt: {receipt}");
```

Este método es esencial para interactuar con contratos inteligentes cuando necesitas realizar cambios en la blockchain, como actualizar el estado o transferir activos.
