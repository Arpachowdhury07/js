# GetBuyWithCryptoStatus

Obtén el estado del intercambio para un hash de transacción devuelto por [BuyWithCrypto](/dotnet/pay/buywithcrypto).

## Uso

```csharp
using Thirdweb.Pay;

var status = await ThirdwebPay.GetBuyWithCryptoStatus(client, txHash);
```

<Details summary="Configuración">

#### Cliente (obligatorio)

`ThirdwebClient`: La instancia del cliente de Thirdweb.

#### Hash de transacción (obligatorio)

`string`: El hash de la transacción del intercambio devuelto por [BuyWithCrypto](/dotnet/pay/buywithcrypto).

</Details>

<Details summary="Valor de retorno">

### Valor de retorno

Un objeto `BuyWithCryptoStatusResult` que contiene las siguientes propiedades:


```csharp
Quote Quote; // El objeto de cotización que contiene los detalles del intercambio.
string SwapType; // El tipo de intercambio, vea el enum SwapType.
TransactionDetails Source; // Los detalles de la transacción de origen.
TransactionDetails Destination; // Los detalles de la transacción de destino.
string Status; // El estado del intercambio, vea el enum SwapStatus.
string SubStatus; // El subestado del intercambio, vea el enum SwapSubStatus.
string FromAddress; // La dirección de origen.
string FailureMessage; // El mensaje de error si el intercambio falló.
string Bridge; // El puente utilizado para el intercambio, si es aplicable.
object PurchaseData; // Datos adicionales pasados al crear la cotización.
```

</Details>
