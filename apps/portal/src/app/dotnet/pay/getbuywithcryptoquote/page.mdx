# GetBuyWithCryptoQuote

Obtén una cotización que contiene un TransactionRequest para intercambiar cualquier par de tokens.

## Uso

```csharp
using Thirdweb.Pay;

var swapQuoteParams = new BuyWithCryptoQuoteParams(
	fromAddress: connectedAddress,
	fromChainId: 137,
	fromTokenAddress: Thirdweb.Constants.NATIVE_TOKEN_ADDRESS; // MATIC
	toTokenAddress: "0x0d500b1d8e8ef31e21c99d1db9a6444d3adf1270"; // WMATIC
	toAmount: "2" // I want to buy 2 WMATIC
);

var quote = await ThirdwebPay.GetBuyWithCryptoQuote(client, swapQuoteParams);
```

<Details summary="Configuración">

#### cliente (requerido)

`ThirdwebClient`: La instancia del cliente de Thirdweb.

#### BuyWithCryptoQuoteParams (requerido)

Un objeto `BuyWithCryptoQuoteParams` (con un constructor útil) que contiene las siguientes propiedades:

```csharp
string fromAddress; // Dirección que ejecuta el intercambio
BigInteger? fromChainId; // ID de la cadena del token de origen
string fromTokenAddress; // Dirección del token de origen
string toTokenAddress; // Dirección del token de destino
string fromAmount; // Opcional, cantidad del token de origen
string fromAmountWei; // Opcional, cantidad del token de origen en wei
BigInteger? toChainId; // Opcional, ID de la cadena del token de destino
string toAmount; // Opcional, cantidad del token de destino
string toAmountWei; // Opcional, cantidad del token de destino en wei
string toAddress; // Opcional, dirección para recibir el token de destino
int? maxSlippageBPS; // Opcional, deslizamiento máximo en puntos básicos
string intentId; // Opcional, identificador de intención usado para vincular el estado a un flujo de BuyWithFiat si lo hubiera
object purchaseData; // Opcional, datos adicionales que se pasarán y retendrán durante el flujo
```

</Details>

```csharp
string quoteId; // Identificador de la cotización

string fromAmountWei; // Cantidad del token desde en wei
string fromAmount; // Cantidad del token desde
string toAmountMinWei; // Cantidad mínima del token hacia en wei
string toAmountMin; // Cantidad mínima del token hacia
string toAmountWei; // Cantidad del token hacia en wei
string toAmount; // Cantidad del token hacia
List<PaymentToken> paymentTokens; // Detalles del token de pago
List<ProcessingFee> processingFees; // Detalles de las tarifas de procesamiento
Estimated estimated; // Valores estimados en USD, deslizamiento, duración si están disponibles
int maxSlippageBPS; // Máximo deslizamiento en puntos básicos

```
