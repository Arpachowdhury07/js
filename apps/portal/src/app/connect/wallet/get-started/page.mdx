# Comienza aquí

thirdweb ofrece múltiples formas de agregar billeteras a tu aplicación. Elige el método que mejor se adapte a tus necesidades:

1. **Billeteras Externas** - Conéctate a billeteras existentes como MetaMask, Coinbase Wallet, etc.
2. **Billeteras In-App** - Crea billeteras con inicio de sesión social/correo electrónico.
3. **Billeteras del Ecosistema** - Crea billeteras con inicio de sesión social/correo electrónico que son compartibles a través de varias aplicaciones.

Estos métodos pueden usarse de manera independiente o juntos en la misma aplicación para proporcionar la mejor experiencia a tus usuarios.

## Obtén tu ID de cliente

Para usar billeteras en tu aplicación, necesitarás un ID de cliente. Puedes obtener uno gratis en [tu panel de control de thirdweb](https://thirdweb.com/create-api-key).
