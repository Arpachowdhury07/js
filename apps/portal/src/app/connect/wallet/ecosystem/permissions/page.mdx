# Gestionar Permisos del Ecosistema

Hay dos opciones de permisos dependiendo de si deseas una billetera de ecosistema abierta o exclusiva. Puedes gestionar estos permisos desde la página de Permisos en tu Panel de Ecosystem Wallet.

<DocImage src={EcosystemWalletPermissions} />

### Cualquiera

Si deseas permitir que cualquier persona integre tu billetera de ecosistema, no se requieren más cambios. Puedes proteger tu ecosistema contra actores maliciosos agregando dominios no deseados a tu lista de bloqueo.

### Lista de Permitidos

Si deseas crear una billetera de ecosistema para socios selectos, puedes dirigirte al panel de Ecosystem Wallet, navegar a Permisos y seleccionar "Lista de Permitidos".

**Agregar Aplicaciones a tu Lista de Permitidos**

Para cada aplicación en tu lista de permitidos, deberás especificar un nombre, una lista de dominios permitidos y un ID de paquete opcional (para aplicaciones móviles). La lista de dominios determinará dónde un socio podrá mostrar tu ecosistema como una opción de inicio de sesión. Una vez que se agregue un socio, se generará un **Partner ID**.

<Callout title="Agregar tu propia aplicación como socio" variant="info">
	Necesitarás agregar tus propios dominios e IDs de paquete a tu lista de permitidos para poder iniciar sesión a través de tu billetera de ecosistema en tu aplicación.
</Callout>
