# Explorador de Billetera del Ecosistema

Cada implementación de Ecosystem Wallet también creará una página de administración de billetera que puedes compartir con tus socios y usuarios. Los usuarios finales pueden visitar esta página para ver tokens y activos en las billeteras dentro de la aplicación que han creado a través de tu ecosistema.

Una vez que despliegues, generaremos y alojaremos automáticamente esta página para ti. Te proporcionaremos una URL que podrás compartir con tus socios del ecosistema y usuarios finales. Podrás encontrar tu URL de explorador en la pestaña Wallet Explorer del [panel de Ecosystem Wallet](https://thirdweb.com/team/~/~/ecosystem).
