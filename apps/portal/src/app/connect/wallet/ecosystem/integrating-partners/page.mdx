# Integrar con Socios

Una de las características clave de las billeteras de ecosistema es la capacidad de permitir que otros desarrolladores creen billeteras integradas en tu nombre. Todos los socios que creen billeteras integradas usando tu billetera de ecosistema accederán a la misma cuenta para tus usuarios finales. Puedes crear socios en el [Panel de Ecosystem Wallet](https://thirdweb.com/team/~/~/ecosystem).

Agregar un nuevo socio creará su **Partner ID** único. Podrán usar este Partner ID para integrar tu billetera de varias maneras:

### SDK de thirdweb

Los socios pueden instalar el SDK de thirdweb y usar el Partner ID que creaste para mostrar el flujo de inicio de sesión con tu marca. Todas las opciones de inicio de sesión en este flujo crearán una billetera integrada que funcionará en todo el ecosistema.

#### Interfaz de Inicio de Sesión

La interfaz de inicio de sesión proporciona un flujo completo con opciones de inicio de sesión por correo electrónico, teléfono, passkey y redes sociales. Las opciones de inicio de sesión social disponibles son determinadas por el creador del ecosistema. Cualquier opción mostrada en el flujo de inicio de sesión creará una billetera integrada dentro del ecosistema especificado.

<Steps>
<Step title='Crear una Billetera'>
Los socios pueden instanciar tu billetera de ecosistema usando tu **Ecosystem ID**.

```ts
import { ecosystemWallet } from "thirdweb/wallets";

// Crear una billetera para un ecosistema "Cualquiera" sin restricciones
const wallet = ecosystemWallet("ecosystem.tu-ecosystem-id");
```

Si configuras tu billetera de ecosistema como Lista de Permitidos, tus socios también deberán especificar su **Partner ID** único (disponible en el Panel de Ecosystem Wallet).

```ts
import { ecosystemWallet } from "thirdweb/wallets";

// Crear una billetera para ecosistemas con "Lista de Permitidos" restringidos a socios
const wallet = ecosystemWallet("ecosystem.tu-ecosystem-id", {
	partnerId: "...",
});
```

</Step>

<Step title='Agregar la Billetera del Ecosistema al Inicio de Sesión'>
Una vez que un socio haya instanciado tu billetera de ecosistema, podrá agregarla a su interfaz de inicio de sesión.

```tsx
import { ConnectButton } from "thirdweb/react";

<ConnectButton
	client={THIRDWEB_CLIENT}
	wallets={[wallet]} // Pasar una billetera sin restricciones o restringida al ConnectButton
	{...props}
/>;
```

</Step>

</Steps>

#### Conector de Billetera con tu marca

Los socios que deseen utilizar tu Ecosystem Wallet como una opción en su flujo de inicio de sesión existente pueden integrar tu billetera por separado. En este caso, solo los inicios de sesión a través de `ecosystem.tu-ecosystem-id` crearán billeteras dentro de tu ecosistema:

<DocImage src={SingleLoginOption} />

```tsx
import { createWallet } from "thirdweb/wallets";

<ConnectButton
	client={THIRDWEB_CLIENT}
	wallets=[
		createWallet("ecosystem.tu-ecosystem-id", {
			partnerId: "...", // Pasar un Partner ID opcional para ecosistemas restringidos
		}),
		createWallet("io.metamask"),
	]
	{...props}
/>;
```
