# Visión general

Un flujo de registro flexible que se adapte a diferentes preferencias es fundamental al incorporar usuarios. thirdweb ofrece múltiples soluciones de billetera para satisfacer diversas necesidades de integración:

1. **Billeteras externas** - Conexión tradicional con más de 350 proveedores
2. **Billetera In-App** - Incorporación sin fricciones con opciones de correo electrónico, redes sociales y claves de acceso
3. **Billetera de Ecosistema** - Servicio de billetera personalizada con marca propia para autenticación entre aplicaciones
