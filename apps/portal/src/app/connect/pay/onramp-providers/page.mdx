# Proveedores de Onramp

thirdweb Pay se integra con Stripe, Kado y Transak para potenciar nuestro onramp. Por defecto, elegimos un proveedor recomendado según la ubicación del usuario, el estado KYC y la moneda.

| Proveedor | Países Soportados                                                                                     |
| --------- | ----------------------------------------------------------------------------------------------------- |
| Transak   | https://transak.notion.site/On-Ramp-Payment-Methods-Fees-Other-Details-b0761634feed4b338a69f4f186d906a5 |
| Kado      | https://www.kado.money/supported-countries                                                            |
| Stripe    | https://docs.stripe.com/crypto/onramp                                                                 |

## Elegir un Proveedor Preferido

Es posible especificar un proveedor de onramp preferido configurando el parámetro `preferredProvider` en el ConnectButton, PayEmbed o Integración Headless. Visita [customizaciones](../pay/customization/connectbutton#preferredProvider) para más detalles, o consulta la [Referencia de API para preferredProvider](https://portal.thirdweb.com/references/typescript/v5/GetBuyWithFiatQuoteParams).
