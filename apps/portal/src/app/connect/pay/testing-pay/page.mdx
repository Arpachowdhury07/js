# Activar Modo de Prueba

Los desarrolladores pueden activar el Modo de Prueba para probar transacciones fiat-a-cripto y cripto-a-cripto en las cadenas de prueba compatibles. Es posible realizar pruebas de ambos casos al mismo tiempo o de forma separada.

## Comprar con Fiat

Al configurar `testMode` como `true` para la opción "Comprar con Fiat", puedes habilitar experiencias de prueba con nuestros proveedores subyacentes (Stripe, Kado y Transak).

Puedes utilizar la siguiente información al usar cualquier proveedor en modo de prueba:

| Provider | Card Type    | Name        | Number              | Expiration Date | CVV          | Other Info                                                       |
| -------- | ------------ | ----------- | ------------------- | --------------- | ------------ | ---------------------------------------------------------------- |
| Kado     | Credit Card  | Test Credit | 4242 4242 4242 4242 | 01/25           | 123          | Test Phone Number: (234) 567-8909                                |
| Kado     | Debit Card   | Test Debit  | 4659 1055 6905 1157 | 01/25           | 123          | Test Phone Number: (234) 567-8909                                |
| Stripe   | Visa         | Any Name    | 4242 4242 4242 4242 | Any Future Date | Any 3 Digits | SSN: 000000000, Address line 1: address_full_match, OTP : 000000 |
| Stripe   | Visa (Debit) | Any Name    | 4000 0566 5566 5556 | Any Future Date | Any 3 Digits | SSN: 000000000, Address line 1: address_full_match, OTP : 000000 |
| Transak  | Visa         | Any Name    | 4111 1111 1111 1111 | 10/33           | 123          | Payment Authorization 3Ds screen password: **Checkout1!**        |

Para las pruebas con Stripe, puedes ver más tarjetas de prueba [aquí](https://docs.stripe.com/testing).

Para las pruebas con Transak, puedes consultar información detallada [aquí](https://docs.transak.com/docs/test-credentials#card-payments).

Para las pruebas con Kado, puedes consultar información detallada [aquí](https://docs.kado.money/integrations/integrate-kado/sandbox).

## Comprar con Criptomonedas

El modo de prueba para "Comprar con Criptomonedas" mostrará solo redes de prueba y monedas de prueba nativas. Si tienes un token personalizado que te gustaría que se admita en modo de prueba, [contáctanos](https://thirdweb.com/contact-us). Puedes ver una lista de las redes de prueba compatibles en la [página de Chainlist de thirdweb](https://thirdweb.com/chainlist?service=pay&type=testnet).
