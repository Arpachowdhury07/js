# Comienza

Aprende cómo agregar onramps y compras de criptomonedas a tu aplicación. Para determinar cuál integración es la adecuada para ti, consulta nuestra [visión general de integraciones](/connect/pay/overview#integration-options).

La siguiente guía utiliza nuestro SDK de React. También puedes aprender cómo integrar Pay con nuestro [SDK de Unity](https://portal.thirdweb.com/unity/pay/getbuywithcryptoquote).

## Option 1: ConnectButton

Pay está disponible de forma predeterminada con nuestro componente `ConnectButton`. Cuando los usuarios inician sesión con Connect, podrán cargar fondos y comprar criptomonedas directamente desde la interfaz de Connect. Puedes leer más sobre `ConnectButton` [en esta guía](/connect/sign-in/ConnectButton).

```tsx
import { ThirdwebProvider, ConnectButton } from "thirdweb/react";

const client = createThirdwebClient({ clientId: your_client_id });

export default function App() {
	return (
		<ThirdwebProvider>
			<ConnectButton client={client} />
		</ThirdwebProvider>
	);
}
```

Nuestro `ConnectButton` tiene varias [opciones de personalización](/connect/pay/customization/connectbutton) para que puedas adaptarlo a tu experiencia.

## Option 2: Embed Pay

El componente `PayEmbed` permite a los usuarios recargar fondos y comprar criptomonedas directamente desde la interfaz de tu aplicación.

```tsx
import { ThirdwebProvider, PayEmbed } from "thirdweb/react";

const client = createThirdwebClient({ clientId: your_client_id });

export default function App() {
	return (
		<ThirdwebProvider>
			<PayEmbed client={client} />
		</ThirdwebProvider>
	);
}
```

El componente integrado de Pay se mostrará de la siguiente manera en la interfaz de tu aplicación:

<DocImage src={GetStartedEmbed} />

¡Y voilà! Tus usuarios ahora pueden agregar fondos y convertir criptomonedas directamente desde tu aplicación.

Nuestro componente `PayEmbed` ofrece varias [opciones de personalización](/connect/pay/customization/payembed) para que lo adaptes a tu experiencia.

## Option 3: Send a Transaction with Pay

Pay está habilitado de forma predeterminada para cualquier llamada a un contrato realizada con `sendTransaction`. Se invocará automáticamente cuando un usuario no tenga suficientes fondos para completar la transacción.

<Callout variant="info">
	ℹ️ `sendTransaction` está disponible en [Connect SDK v5](/typescript/v5) o versiones superiores.
</Callout>


```tsx
import { useSendTransaction } from "thirdweb/react";
import { mintTo } from "thirdweb/extensions/erc721";

const { mutate: sendTx, data: transactionResult } = useSendTransaction();

const onClick = () => {
	const transaction = mintTo({
		contract,
		to: "0x...",
		nft: {
			name: "NFT Name",
			description: "NFT Description",
			image: "https://example.com/image.png",
		},
	});
	sendTx(transaction);
};
```

Cuando un usuario haga clic en este botón, Pay realizará una estimación de gas para determinar si el usuario tiene un saldo suficiente. Si su saldo es suficiente, la transacción se ejecutará normalmente. Si su saldo no es suficiente, aparecerá el siguiente modal solicitando al usuario que agregue fondos o convierta criptomonedas a la cantidad requerida:

<DocImage src={GetStartedSend} />

Una vez que el usuario agregue fondos o convierta sus criptomonedas al token requerido, se le pedirá nuevamente que confirme la transacción. La transacción se ejecutará como se espera.

Para una personalización más profunda del modal de transacciones de Pay, puedes consultar nuestra [guía de personalización de sendTransaction](/connect/pay/customization/send-transaction).
