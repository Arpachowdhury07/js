# Abstracción de Cuentas

Todo lo que necesitas para aprovechar la tecnología de abstracción de cuentas y habilitar experiencias de usuario sin interrupciones para tus usuarios. Obtienes todas las herramientas para integrar la abstracción de cuentas en tu aplicación. Esto incluye:

- Contratos de fábrica de cuentas que te permiten crear cuentas inteligentes para tus usuarios.
- Bundler, que es la infraestructura necesaria para procesar transacciones de abstracción de cuentas (conocidas como User Ops).
- Paymaster, que te permite patrocinar tarifas de transacción para tus usuarios.
