# Infraestructura de Bundler y Paymaster

El SDK de thirdweb maneja toda la carga pesada de las operaciones de bundling y de cubrir las tarifas de gas con una infraestructura lista para usar.
Los servicios de paymaster y bundler de thirdweb son **independientes** y pueden ser utilizados con cualquier contrato de cuenta inteligente sin usar los SDKs de thirdweb.

En las testnets, el único requisito es obtener un ID de cliente gratuito para comenzar.
Una vez que estés listo para desplegar en mainnets, necesitarás una [cuenta de facturación activa](https://thirdweb.com/team/~/~/settings/billing).
Puedes configurar tu ID de cliente para restringir las interacciones solo con tus propios contratos o con cualquier contrato.

## Usando un Bundler Personalizado

Las cuentas inteligentes son libres de usar con tu propia infraestructura de abstracción de cuentas. Para usar tu propio paymaster y bundler,
pasa el `bundlerUrl` en las sobreescrituras de `SmartWalletOptions`:


```ts
const config: SmartWalletOptions = {
	chain,
	sponsorGas: true,
	overrides: {
		bundlerUrl: "your-bundler-url",
	},
};
```

## Usando un Paymaster Personalizado

También puedes proporcionar una lógica de paymaster completamente personalizada al proporcionar una función `paymaster` en las sobreescrituras de `SmartWalletOptions`.

```ts
const config: SmartWalletOptions = {
	chain,
	sponsorGas: true,
	overrides: {
		paymaster: async (userOp: UserOperation) => {
			// your custom paymaster logic
			return myPaymasterResultLogic(userOp);
		},
	},
};
```

## Precios y Facturación

Para usar la infraestructura de abstracción de cuentas de thirdweb, necesitas configurar una cuenta de facturación en la página de Configuración del panel de thirdweb.

**Precios**:

- **Bundler**: Las llamadas de transacción en bundles (llamadas no patrocinadas) son gratuitas.
- **Paymaster**: Prima del 10% sobre la tarifa de red basada en tu uso.

Encuentra más información sobre los diferentes niveles de facturación visitando [la página de precios de thirdweb](https://thirdweb.com/pricing).
