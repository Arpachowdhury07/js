# Leer el estado del contrato

La forma recomendada de leer el estado del contrato es usar la función `readContract` y pasar la firma del método Solidity y los parámetros. Esto es seguro en cuanto a tipos, basado en la firma del método Solidity que defines. Puedes obtener la firma del método del contrato que desees directamente desde el código Solidity.

```ts
import { readContract } from "thirdweb";

const balance = await readContract({
	contract: contract,
	method: "function balanceOf(address) view returns (uint256)",
	params: ["0x123..."],
});
```

Esto ejecutará la lectura inmediatamente y devolverá el resultado desde la blockchain.

## Leer eventos del contrato

La forma recomendada de leer los eventos del contrato es usar la función `getContractEvents` y pasar un evento preparado con la firma del evento Solidity y los parámetros. Esto es seguro en cuanto a tipos, basado en la firma del evento Solidity que defines. Puedes obtener la firma del evento del contrato que desees directamente desde el código Solidity.

```ts
import { getContractEvents, prepareEvent } from "thirdweb";

const myEvent = prepareEvent({
  signature: "event Transfer(address indexed from, address indexed to, uint256 value)",
});

const events = await getContractEvents({
  contract: myContract,
  events: [myEvent],
  blockRange: 1000,
});
```

### Generar todas las funciones de lectura y eventos para un contrato desplegado

Usando la CLI, puedes generar funciones optimizadas para todas las llamadas posibles a un contrato. Esto te ahorra tiempo y precomputará toda la codificación necesaria.

```shell
npx thirdweb generate <contractId>/<contractAddress>
```

Lee más sobre cómo [generar funciones de extensión usando la CLI](/typescript/v5/extensions/generate).
