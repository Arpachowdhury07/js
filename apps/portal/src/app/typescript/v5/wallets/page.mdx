# Cuentas y Billeteras

En el SDK de thirdweb distinguimos entre "cuentas" y "billeteras". Creemos que esto proporciona una API más predecible y flexible para los desarrolladores.

## ¿Qué es una Cuenta?

- Una cuenta siempre tiene una `dirección` y una manera de `firmar` mensajes, transacciones y datos tipados.
- Una cuenta siempre está mapeada a exactamente una dirección en la blockchain.
- Una cuenta no puede ser "conectada" o "desconectada" como una billetera, en su lugar, a menudo es el resultado de una billetera conectada.

Ver también: [Cuenta (ethereum.org)](https://ethereum.org/en/glossary/#account)

## ¿Qué es una Billetera?

- Una billetera "contiene" una o más cuentas.
- Una billetera puede ser "conectada" (a menudo pidiendo la aprobación del usuario) o "desconectada".
- Una billetera no puede firmar mensajes, transacciones o datos tipados de manera independiente, en su lugar, delega esto a la(s) cuenta(s) que "contiene".

## Ejemplo: Conectar una billetera y acceder a una cuenta para enviar una transacción.

```ts
import { sendTransaction } from "thirdweb";
// Usamos MetaMask como ejemplo, el patrón es el mismo para todas las billeteras
import { createWallet } from "thirdweb/wallets";

// Inicializa la billetera, puedes elegir cualquiera de los 300+ conectores de billetera soportados
// Los IDs de billetera están tipados, deja que tu editor TS los complete automáticamente
// ej: "io.metamask", "com.coinbase.wallet", "me.rainbow", etc...
const wallet = createWallet("io.metamask");

// Conecta la billetera, esto devuelve una promesa que se resuelve con la cuenta conectada
const account = await wallet.connect({
	// pasa el cliente que creaste con `createThirdwebClient()`
	client,
});

// Firma y envía una transacción con la cuenta -> devuelve el hash de la transacción
const { transactionHash } = await sendTransaction({
	// suponiendo que hayas llamado a `prepareTransaction()` o `prepareContractCall()` antes, lo que devuelve la transacción preparada para enviar
	transaction,
	// Pasa la cuenta para firmar la transacción
	account,
});
```
