## Características

- **Componentes UI para React y React Native**: para la conexión de billeteras, transacciones y más
- **Billeteras en la app**: soporte de primera clase para inicios de sesión por email y redes sociales
- **Abstracción de cuentas**: soporte de primera clase para cuentas inteligentes ERC4337
- **API de contratos segura para tipos**: completamente tipada con ABI legible por humanos
- **Generación de código**: CLI para generar contratos optimizados y interfaces seguras para tipos
- **RPC para cualquier red EVM**: RPCs de alto rendimiento solo con el ID de la red
- **Subida/descarga de IPFS**: integración simple y eficiente de IPFS
- **Resolución automática de ABI**: resuelve ABIs para cualquier contrato desplegado solo con la dirección
- **Interoperabilidad con Ethers / Viem**: adaptadores para las bibliotecas ethers y viem

## Principios de diseño

- **Un solo paquete:** para Node, React y React Native con tree shaking para un tamaño mínimo.
- **Optimizado para el rendimiento:** enfocado en velocidad y eficiencia con principios de responsabilidad única.
- **Infraestructura lista para usar:** acceso directo a toda la infraestructura de thirdweb desde billeteras hasta RPCs y IPFS.
- **Experiencia para desarrolladores:** segura para tipos, fácil de usar, con patrones repetibles y predecibles.
- **Interoperable:** funciona con cualquier proveedor de infraestructura y con otras bibliotecas como viem y ethers.
