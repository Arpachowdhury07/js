# Insight

Insight es una herramienta poderosa que te permite recuperar datos de blockchain de cualquier cadena EVM, enriquecerlos con metadatos y transformarlos utilizando lógica personalizada. Ya sea que estés construyendo un sistema de inventario de juegos, rastreando métricas DeFi o analizando colecciones de NFT, Insight facilita la obtención de los datos que necesitas con simples llamadas a la API.

## ¿Por qué usar Insight?

- Le da a los desarrolladores una API de datos fácilmente comprensible para consultar datos de blockchain.
- No es necesario indexar blockchains por ti mismo ni gestionar los costos de infraestructura y RPC.
- Transforma y enriquece los datos con Blueprints personalizados.
- Insight es de código abierto.
