# Referencia de CLI

Usa el CLI para desplegar tus contratos directamente a cualquier blockchain compatible con EVM ejecutando:

```bash
npx thirdweb [options]
```

### Opciones

```bash
-p--path <string>
```

- Ruta al proyecto

```bash
-c--clean
```

- Limpiar la caché antes de construir

```bash
--dry-run
```

- Ejecución de prueba (omitir la publicación real)

```bash
-d--debug
```

- Mostrar los logs de depuración

```bash
--ci
```

- Modo de Integración Continua

```bash
--dist-path <string>
```

- Ruta a la carpeta dist donde está la aplicación basada en HTML

```bash
-n--name <string>
```

- Nombre del contrato preconstruido o publicado (como nft-drop)

```bash
-f--file <string>
```

- Filtrar archivos de contrato que contengan este nombre de archivo

```bash
-cn--contract-name <string>
```

- Filtrar contratos que contengan este nombre de contrato

```bash
-cv--contract-version <string>
```

- Versión del contrato publicado

```bash
--app
```

- Desplegar una aplicación web en almacenamiento descentralizado

```bash
--contract
```

- Desplegar un contrato inteligente en blockchains

```bash
--dynamic
```

- Desplegar un contrato inteligente dinámico compuesto por extensiones en blockchains

```bash
-k--key <string>
```

- Parámetro opcional para usar la clave secreta proporcionada, útil para casos de uso de integración continua.
