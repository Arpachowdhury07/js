# Desplegar Contrato

Para desplegar un contrato usando Deploy:

<Steps>

<Step title="Ejecutar CLI">

Navega a la raíz del repositorio del contrato inteligente usando el CLI. Luego, ejecuta el comando `npx thirdweb deploy`.

<DocImage src={deployCLI} alt="Captura de pantalla de la salida del comando en el CLI" />

</Step>

<Step title="Autorizar dispositivo">

Para desplegar desde una billetera deseada en el navegador, conecta y autoriza el dispositivo.

<DocImage src={linkDevice} alt="Captura de pantalla para vincular el dispositivo" />

</Step>

<Step title="Rellenar parámetros">

Una vez completado, se abrirá una interfaz de usuario en el navegador que permite rellenar los parámetros del contrato.

<DocImage
	src={detectedExtensions}
	alt="Captura de pantalla de las extensiones detectadas al desplegar contratos"
/>

</Step>

<Step title="Seleccionar opciones">

Rellena los campos necesarios, luego busca el menú desplegable de Red / Cadena para seleccionar una red para el despliegue. Elige la opción `Agregar al dashboard` si deseas gestionar este contrato desde el dashboard y selecciona Desplegar ahora. Esto requerirá firmar una transacción en la billetera conectada.

<DocImage
	src={deployContract}
	alt="Captura de pantalla de la confirmación del despliegue del contrato"
/>

</Step>

<Step title="Gestionar contrato">

Una vez desplegado, puedes gestionar tu contrato a través del dashboard.

</Step>

</Steps>
