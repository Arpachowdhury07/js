# Deploy

Deploy es una herramienta para desplegar cualquier contrato en cualquier cadena compatible con EVM a través de un solo comando y una interfaz fácil de usar. Deploy reduce las complejidades del despliegue de contratos al eliminar pasos como la codificación de claves privadas, la gestión de URLs de RPC o la ejecución de scripts para desplegar tus contratos.

Simplemente ejecuta el siguiente comando en la raíz de tu directorio de contratos:

```bash
npx thirdweb deploy
```
