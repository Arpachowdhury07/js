# Referencia CLI de Publicación

Usa la CLI para publicar tus contratos directamente en cualquier cadena compatible con EVM utilizando el panel de control:

```bash
npx thirdweb publish
```

Este comando hará lo siguiente:

- Compilará tu contrato inteligente.
- Subirá el código fuente del contrato (ABI) a IPFS.
- Abrirá el flujo de publicación en el panel de control.

### Opciones

```bash
-p--path <string>
```

- Ruta al proyecto

```bash
-f--file <string>
```

- Filtrar archivos de contrato que contengan este nombre de archivo

```bash
-cn--contract-name <string>
```

- Filtrar contratos que contengan este nombre de contrato

```bash
-c--clean
```

- Limpiar la caché antes de construir

```bash
--dry-run
```

- Simulación (omitir la publicación real)

```bash
-d--debug
```

- Mostrar registros de depuración

```bash
--ci
```

- Modo de Integración Continua

```bash
-k--key <string>
```
