# Publicar

Usa la [CLI](/cli) para publicar tus contratos directamente en **cualquier cadena compatible con EVM** usando el [panel de control](https://thirdweb.com/team):

```bash
npx thirdweb publish
```

Este comando hará lo siguiente:

- Compilará tu contrato inteligente.
- Subirá el código fuente del contrato (ABI) a IPFS.
- Abrirá el flujo de publicación en el panel de control.

## Cómo publicar tu contrato

En tu navegador, sigue los pasos a continuación para publicar tu contrato:

### Configura los metadatos del contrato

Desde el panel de control, proporciona los metadatos de tu contrato inteligente, incluyendo:

- **Nombre**
- **Imagen**
- **Descripción**
- **README** - Se admite Markdown. Lo que hace tu contrato, casos de uso comunes y ejemplos son cosas excelentes para incluir aquí.

### Agrega una versión e informe de auditoría (opcional)

Usa [versionado semántico](https://semver.org/) para publicar de manera incremental nuevas versiones de tu contrato inteligente;
ideal cuando deseas hacer cambios en tu contrato sin romper integraciones existentes.

Cada versión de tu contrato inteligente se almacena en la cadena y puede ser desplegada por cualquiera. 
Puedes incluir notas de la versión, para describir los cambios realizados entre versiones.

También puedes vincular cada versión a un informe de auditoría. Puedes pegar un enlace a este, o arrastrar y soltar un PDF. Nosotros generaremos un enlace IPFS para ti.

### Elige cómo los usuarios desplegarán tu contrato publicado

Selecciona el tipo de contrato para tu contrato publicado:

- **Despliegue directo**: Elige esta opción si tu contrato no sigue el patrón de contrato actualizable.
  - Esta opción se puede usar para **cualquier contrato**.
  - No hay requisitos para usar esta opción para publicar tu contrato.
  - Los usuarios desplegarán el **contrato completo**: almacenamiento y lógica de implementación.
- [Desplegar a través de Factory](/contracts/publish/publish-options): Elige esta opción si deseas que los usuarios desplieguen proxies de tu contrato usando el contrato factory predeterminado de thirdweb o usando tu propia factory personalizada ya desplegada.
  - [Factory predeterminado](/contracts/publish/publish-options#default-factory) - Usa esta opción si tu contrato sigue el patrón de contrato actualizable.
  - [Factory personalizada](/contracts/publish/publish-options#custom-factory) - Usa esta opción para usar tu propia factory personalizada predesplegada para crear proxies para tus usuarios.

Una vez seleccionada, haz clic en "Siguiente" para continuar con el flujo correspondiente al tipo de contrato seleccionado.

### Elige en qué(s) red(es) publicar tu contrato

Selecciona la(s) red(es) en las que te gustaría publicar tu contrato. Puedes seleccionar múltiples redes o cualquier EVM.

Si elegiste publicar mediante despliegue directo, se te pedirá que selecciones las redes en las que deseas que los usuarios puedan desplegar tu contrato y luego haz clic en el botón "Siguiente".
Si elegiste publicar mediante factory, se te pedirá que selecciones las redes en las que deseas que los usuarios puedan desplegar tu contrato al configurar los "Ajustes de despliegue de Factory".

### Personaliza los parámetros del contrato

Ayuda a los usuarios que intentan desplegar tu contrato inteligente cambiando la apariencia de los parámetros de tu contrato. Estos son los campos que los usuarios deben completar antes de desplegar tu contrato, y cada campo corresponde a la variable asociada en el constructor de tu contrato.

- **Nombre para mostrar**: Usa esto para tener una versión más legible del nombre del parámetro. De lo contrario, se mostrará el nombre del parámetro tal como está en el contrato.
- **Valor predeterminado**: Esto se pre-rellenará en el formulario de despliegue. Usa esto para guiar a los usuarios sobre qué valor ingresar. Si el parámetro es de tipo dirección, puedes pre-rellenar la dirección del wallet conectado estableciendo el valor predeterminado como tu wallet conectado.
- **Descripción**: Usa esto para aclarar cualquier confusión y explicar qué valor se espera para este parámetro.
- **Avanzado**: Marca esta opción si deseas ocultar este parámetro del formulario de despliegue. Esto es útil para parámetros que no se necesitan cambiar por el usuario. Esta opción solo está disponible para parámetros que tienen un valor predeterminado.

### Publicar contrato

Finalmente, haz clic en el botón "Publicar contrato" para publicar tu contrato inteligente en el registro en cadena.
