# Airdrop ERC-20

Airdrop de tokens ERC-20 o el token nativo de la cadena (por ejemplo, ether para la red principal de Ethereum) a una lista de destinatarios.

El contrato `AirdropERC20` es adecuado para cuando deseas transferir tokens ERC20 o tokens nativos a una lista de direcciones de destinatarios, donde no se espera que los destinatarios reclamen individualmente su airdrop; simplemente lo reciben en una transferencia.
Al hacer un airdrop de tokens con este contrato, debes especificar:

- La dirección del token ERC20 que se está transfiriendo. (Para tokens nativos, la dirección utilizada es `0xEeeeeEeeeEeEeeEeEeEeeEEEeeeeEeeeeeeeEEeE`)
- La dirección desde la que el token será transferido a la lista de destinatarios.
- Un array de objetos que contiene la dirección del destinatario y las cantidades de tokens, por ejemplo:


```typescript
[
	{
		recipient: "0x123...",
		amount: 1, // these amounts are in wei
	},
	{
		recipient: "0xabc...",
		amount: 10, // these amounts are in wei
	},
];
```

nota: el propietario de los tokens debe aprobar sus tokens a este contrato de airdrop, llamando a la función relacionada con la aprobación en el contrato del token ERC20.

**Costos estimados de gas para hacer un airdrop de tokens:**
Costo de hacer un airdrop de `1` token ERC20: ~`66,600` gas. Costo de hacer un airdrop de cada token ERC20 adicional: ~`32,526` gas.
