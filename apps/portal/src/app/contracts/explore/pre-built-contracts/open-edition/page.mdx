# Edición Abierta ERC721

El contrato `OpenEditionERC721` es una colección de NFTs ERC721 de 'edición abierta'. El contrato utiliza el estándar [ERC-721A](/contracts/build/extensions/erc-721/ERC721).

El contrato `OpenEditionERC721` es ideal cuando deseas liberar una edición abierta de NFTs como la colección Base, `Introduced`.

Todos los NFTs en el contrato tienen metadatos compartidos, excepto que cada NFT tiene su ID de token único añadido al nombre del NFT. Un administrador puede configurar estos metadatos compartidos en cualquier momento.

El contrato no requiere que el administrador establezca un límite en la cantidad total de NFTs. El administrador puede establecer fases de reclamación (como en NFT Drop) para aplicar restricciones en la acuñación de NFTs, como un precio, horarios de inicio y fin de acuñación, etc.
