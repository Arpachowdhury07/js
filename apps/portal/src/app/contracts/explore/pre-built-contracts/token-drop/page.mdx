# Token Drop

El contrato de Token Drop es una forma de liberar tus tokens <a href="https://eips.ethereum.org/EIPS/eip-20">ERC20</a>
a un precio fijo.

Te permite definir las condiciones sobre cuándo y cómo tus usuarios pueden reclamar
tus tokens; incluyendo listas de permitidos, fechas de lanzamiento y límites de reclamación.

En el Token Drop, defines el precio de tus tokens en cada fase de reclamación y puedes establecer un
límite sobre cuántos tokens deseas liberar. Otros usuarios podrán reclamar tus tokens bajo
las condiciones que hayas definido.

<Callout title="Diseño del Drop" variant="info">

## Casos de uso y ejemplos

Podrías usar el contrato de Token Drop para:

- Liberar tu nueva criptomoneda a un precio fijo, como 1 MATIC por token.
- Permitir que un conjunto específico de carteras reclamen tus tokens ERC20 antes de liberarlos al público.
- Permitir que los usuarios reclamen tus tokens hasta una fecha específica.
