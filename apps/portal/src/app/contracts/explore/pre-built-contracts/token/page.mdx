# Token

El contrato de Token está diseñado para crear una moneda digital y es compatible con el estándar [ERC20](https://eips.ethereum.org/EIPS/eip-20).
Puede ser comprado e intercambiado entre usuarios en un exchange, utilizado para comprar y vender NFTs en un marketplace, etc.

A diferencia del contrato de [Token Drop](/contracts/explore/pre-built-contracts/token-drop), el contrato de Token no permite a los usuarios reclamar tokens bajo los
criterios de las condiciones de reclamación.

## Casos de uso y ejemplos

Podrías usar el contrato de Token para:

- Crear tu propia criptomoneda
- Vender tus NFTs en el [Marketplace](/contracts/explore/pre-built-contracts/marketplace) en tu token personalizado
- Recompensar a los usuarios con tokens por alguna acción que realicen
