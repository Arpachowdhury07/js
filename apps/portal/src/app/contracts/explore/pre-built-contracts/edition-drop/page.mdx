# Edition Drop

El contrato **Edition Drop** es ideal cuando deseas liberar muchos NFTs con los mismos metadatos que los usuarios puedan reclamar. Este contrato sigue la extensión [ERC-1155](https://eips.ethereum.org/EIPS/eip-1155), también conocida como "Tokens Semi-Fungibles".

El contrato Edition Drop te permite definir las condiciones de cuándo y cómo tus usuarios pueden reclamar un NFT, que ha sido previamente [acuñado de forma perezosa](/glossary/lazy-minting) por el administrador del contrato (usualmente el desplegador), incluyendo listas permitidas, fechas de lanzamiento y el número máximo de NFTs que pueden ser reclamados.

## Extensiones

Este contrato incluye las siguientes extensiones:

- [ERC1155](/contracts/build/extensions/erc-1155/ERC1155)
- [ERC1155Supply](/contracts/build/extensions/erc-1155/ERC1155Supply)
- [ERC1155Enumerable](/contracts/build/extensions/erc-1155/ERC1155Enumerable)
- [ERC1155LazyMintable](/contracts/build/extensions/general/LazyMintable)
- [ERC1155ClaimPhases](/contracts/build/extensions/erc-1155/ERC1155ClaimPhases)
- [Royalty](/contracts/build/extensions/general/Royalty)
- [PlatformFee](/contracts/build/extensions/general/PlatformFee)
- [PrimarySale](/contracts/build/extensions/general/PrimarySale)
- [Permissions](/contracts/build/extensions/general/Permissions)
- [PermissionsEnumerable](/contracts/build/extensions/general/PermissionsEnumerable)
- [ContractMetadata](/contracts/build/extensions/general/ContractMetadata)
- [Ownable](/contracts/build/extensions/general/Ownable)

## Casos de uso y ejemplos

Podrías usar el contrato Edition Drop para:

- Crear membresías NFT como nuestras [Tarjetas de Acceso Anticipado](https://opensea.io/collection/thirdweb-community) que quieres que tus usuarios reclamen
- Liberar un artículo en tu juego por un tiempo limitado
- Crear 100 NFTs basados en una obra de arte, y permitir que los usuarios reclamen uno por billetera

## Crear y configurar Edition Drop

Aprende cómo crear y configurar tu contrato inteligente utilizando el [panel](https://thirdweb.com/team).

### Crear un contrato Edition Drop

Despliega el contrato Edition Drop en cualquier red EVM utilizando el botón a continuación.

### Acuñar NFTs de forma perezosa

Acuñar de forma perezosa es el proceso de cargar los metadatos de tus NFT(s) sin acuñarlos _todavía_.

Esto es útil cuando deseas que _otras_ billeteras acuñen NFTs de tu drop en sus propias billeteras.

Para acuñar de forma perezosa un NFT(s), haz clic en **Crear** para acuñar de forma perezosa un solo NFT, o **Subida en Lote** para acuñar de forma perezosa múltiples NFTs a la vez.

Para comenzar, puedes usar uno de nuestros archivos de ejemplo para [CSV](https://thirdweb.com/assets/examples/example.csv) o [JSON](https://thirdweb.com/assets/examples/example.json) de metadatos.

### Configurar Fases de Reclamación

En el contrato Edition Drop, configuras las fases de reclamación para cada NFT, haciendo clic en el botón **Claim Phases**.

Una fase de reclamación es un conjunto de condiciones que definen cuándo y cómo tus usuarios pueden reclamar un NFT de tu drop.

**El contrato Edition Drop admite múltiples fases de reclamación secuenciales**.

Dentro de una fase de reclamación, puedes definir:

- Cuándo comenzará la fase de reclamación
- Cuántos NFTs deseas liberar
- Cuánto deseas cobrar por NFT
- Qué moneda deseas utilizar
- Qué direcciones de billetera pueden acuñar (lista permitida)
- Cuántos NFTs se pueden reclamar por transacción
- Cuántos segundos deben esperar las billeteras entre reclamaciones

Para agregar una fase de reclamación a un token, haz clic en **Agregar Fase de Reclamación Inicial**:

### Airdrop de NFTs

Haz clic en **Airdrop** en el token que deseas usar, y sube un archivo `.csv` que contenga una lista de direcciones a las que deseas hacer el airdrop de NFTs.

En el archivo `.csv`, debes especificar una columna `address` y una columna `quantity`, representando la dirección a la que deseas enviar los NFTs y cuántos deseas enviar.
Puedes descargar nuestro ejemplo de [Archivo CSV](https://thirdweb.com/assets/examples/airdrop.csv) para comenzar.

Para configurar la imagen, nombre y descripción del contrato, haz clic en la pestaña **Configuración**:
