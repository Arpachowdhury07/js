# Airdrop ERC-721

Airdrop de NFTs ERC721 a una lista de destinatarios.

El contrato `AirdropERC721` es adecuado para usar cuando deseas transferir NFTs ERC721 a una lista de direcciones de destinatarios, donde no se espera que los destinatarios reclamen individualmente su airdrop; simplemente lo reciben en una transferencia.
Al hacer un airdrop de NFTs con este contrato, debes especificar:

- La dirección de la colección de NFTs desde la cual se están transfiriendo los NFTs.
- La dirección del propietario de los NFTs desde donde los NFTs serán transferidos a la lista de destinatarios.
- Un array de objetos que contiene la dirección del destinatario y los token-ids, por ejemplo:


```typescript
[
	{
		recipient: "0x123...",
		tokenId: 1,
	},
	{
		recipient: "0xabc...",
		tokenId: 2,
	},
];
```

nota: el propietario de los tokens debe aprobar sus tokens a este contrato de airdrop, llamando a la función relacionada con la aprobación en el contrato ERC721.

**Costos estimados de gas para hacer un airdrop de tokens:**
Costo de hacer un airdrop de `1` token ERC721: ~`76,521` gas. Costo de hacer un airdrop de cada token ERC721 adicional: ~`44,610` gas.
