# Multiwrap

El contrato **Multiwrap** te permite envolver tokens [ERC20](https://eips.ethereum.org/EIPS/eip-20),
[ERC721](https://eips.ethereum.org/EIPS/eip-721) y
[ERC1155](https://eips.ethereum.org/EIPS/eip-1155)
en un **nuevo** NFT ERC721 envuelto.
El NFT envuelto puede ser desenrollado (quemado) de nuevo en los tokens subyacentes.

El propio NFT envuelto también actúa como un token ERC721, que puede tener los metadatos que esperarías de un NFT,
como nombre, imagen, descripción, etc.

## Casos de uso y ejemplos

Puedes usar el contrato Multiwrap para:

- Combinar múltiples NFTs en un solo NFT con nuevos metadatos
- Envolver NFTs junto con tokens ERC20 para intercambiarlos fácilmente por otros NFTs
- Crear un "paquete" transparente en tu juego, donde el contenido del paquete es visible para el jugador
