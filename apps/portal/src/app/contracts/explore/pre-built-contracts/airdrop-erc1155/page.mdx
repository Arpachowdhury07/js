# Airdrop ERC1155

Airdrop de NFTs ERC1155 a una lista de destinatarios.

El contrato `AirdropERC1155` es adecuado para usar cuando deseas transferir NFTs ERC1155 a una lista de direcciones de destinatarios, donde no se espera que los destinatarios reclamen individualmente su airdrop; simplemente lo reciben en una transferencia.
Al hacer un airdrop de tokens con este contrato, debes especificar:

- La dirección de la Edición (contrato ERC-1155) desde la cual se están transfiriendo los NFTs.
- La dirección del propietario de los NFTs desde donde los NFTs serán transferidos a la lista de destinatarios.
- Un array de objetos que contiene la dirección del destinatario, los token-ids y las cantidades, por ejemplo:

```typescript
[
	{
		recipient: "0x123...",
		tokenId: 1,
		amount: 5,
	},
	{
		recipient: "0xabc...",
		tokenId: 1,
		amount: 5,
	},
];
```

nota: el propietario de los tokens debe aprobar sus tokens a este contrato de airdrop, llamando a las funciones relacionadas con la aprobación en el contrato ERC1155.

**Costos estimados de gas para hacer un airdrop de tokens:**
Costo de hacer un airdrop de `1` token ERC1155: ~`51,571` gas. Costo de hacer un airdrop de cada token ERC1155 adicional: ~`42,672` gas.
