# Airdrop ERC1155 Reclamable

El contrato `AirdropERC1155Claimable` es adecuado para usar cuando deseas hacer un airdrop de NFTs ERC1155 a una lista de direcciones de destinatarios, donde estos destinatarios deben reclamar individualmente su propio airdrop.

Al desplegar este contrato, debes especificar:

- La dirección de los NFTs ERC1155 que se están haciendo airdrop.
- Los tokenIds de los NFTs a distribuir.
- Las cantidades de cada tokenId a distribuir.
- La dirección del propietario de los NFTs desde donde los NFTs serán transferidos en el airdrop.
- La marca de tiempo UNIX después de la cual el período de airdrop expirará.
- Una cantidad máxima reclamable por billetera (para reclamar el airdrop) para cada tokenId.
- Un merkle root opcional para la lista permitida de destinatarios que pueden reclamar el airdrop, para cada tokenId.

Al reclamar un airdrop, un destinatario debe especificar:

- La dirección a la que transferir el airdrop reclamado.
- El tokenId del NFT a reclamar.
- La cantidad total a reclamar.
- La prueba de inclusión en el merkle root de la lista permitida, si existe.
