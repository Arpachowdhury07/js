# NFT Drop

NFT Drop permite a los usuarios liberar una colección de NFTs únicos utilizando el estándar ERC721A. NFT Drop utiliza un mecanismo de “drop” diseñado para colecciones grandes para distribuir NFTs.

<Callout title="NFT Drop vs NFT Collection" variant="info">
	NFT Drop se diferencia de [NFT
	Collection](/contracts/explore/pre-built-contracts/nft-collection) en que
	permite a los usuarios acuñar los NFTs de forma perezosa directamente en su billetera. Esto elimina
	la necesidad de acuñar el NFT en la billetera del creador y luego transferir el activo
	a la billetera de otro usuario.
</Callout>

El contrato NFT Drop se puede desplegar a través del [panel](/contracts/explore/pre-built-contracts/nft-drop#deploy-nft-drop-dashboard), [CLI](/contracts/explore/pre-built-contracts/nft-drop#deploy-nft-drop-cli), o [SDKs](/contracts/explore/pre-built-contracts/nft-drop#deploy-nft-drop-sdk).

<Callout title="Diseño del Drop" variant="info">
	Aprende más sobre por qué se construyeron los contratos Drop, cómo fueron diseñados y
	sus limitaciones leyendo el [Documento de Diseño del Drop](/contracts/design-docs/drop)
</Callout>

## Casos de uso

- Crear NFTs de acceso restringido que solo puedan ser reclamados por una lista de billeteras especificadas
- Crear grandes colecciones de imágenes de perfil NFT únicas, donde cada NFT corresponda a metadatos o características únicas
- Crear NFTs de obras de arte destinados para distribución

## Desplegar NFT Drop (Panel)

Aprende cómo desplegar y administrar un contrato NFT Drop utilizando el panel.

### Desplegar y configurar el contrato

<Steps>

<Step title="Seleccionar Contrato">
	Navega a Contratos > Explorar > [NFT
	Drop](https://thirdweb.com/thirdweb.eth/DropERC721)
</Step>

<Step title="Desplegar">

Selecciona `Desplegar ahora`

<DocImage src={deployNowImage} alt="Vista de desplegar un contrato desde Explorar" />

</Step>

<Step title="Configurar Metadatos">

Aparecerá un modal para configurar los Metadatos del Contrato o ajustes para organizar y distinguir entre contratos. Configura los siguientes metadatos:

- **Nombre** - nombre del contrato.
- **Símbolo (opcional)** - nombre corto o símbolo para denotar un grupo específico de tokens. Similar a un ticker de acciones.
- **Descripción (opcional)** - descripción del contrato.
- **Imagen (opcional)** - imagen o ícono del contrato

<DocImage src={contractMetadataImage} alt="Ajustes de metadatos del contrato" />

</Step>

<Step title="Distribución de fondos">

Configura los destinatarios de los fondos para Ventas Primarias y Ventas Secundarias pasando las direcciones de billeteras correspondientes que deseas que reciban los fondos.

- **Ventas Primarias** - El destinatario inicial recibirá los fondos de las ventas realizadas al acuñar inicialmente el NFT.

- **Regalías / Ventas Secundarias** - El destinatario que recibirá los fondos de cualquier venta secundaria o reventa del NFT.

  <DocImage src={fundsImage} alt="Ajustes de distribución de fondos" />

- **(Opcional) Tarifas de Plataforma** - El destinatario que recibirá tarifas adicionales por todas las ventas primarias que ocurran en este contrato. (Esto es útil si estás desplegando este contrato para un tercero y deseas cobrar tarifas por tu servicio).

  <DocImage
  	src={advancedConfigurationImage}
  	alt="Captura de pantalla de configuración avanzada"
  />

</Step>

<Step title="Elegir red">

Elige una red / cadena para desplegar. [Ver lista de redes disponibles.](https://thirdweb.com/chainlist)

<DocImage src={chooseNetworkImage} alt="Pantalla para elegir red" />

</Step>

<Step title="Firmar transacción">
	Selecciona `Desplegar` y firma la transacción solicitada en la billetera conectada para
	finalizar el despliegue del contrato.
</Step>

</Steps>

### Subir NFTs

**Subida Única**

- La subida única es una opción preferida para usuarios que suben pocos NFTs a la vez.

**Subida en Lote**

- La subida en lote es un método que permite subir grandes cantidades de NFTs con un solo clic.

## Desplegar NFT Drop (CLI)

1. En el CLI ejecuta el siguiente comando de despliegue:

   ```bash
   npx thirdweb deploy -n --name nft-drop
   ```

2. Sigue las indicaciones para completar tu contrato siguiendo la sección Desplegar NFT Drop (Panel).
