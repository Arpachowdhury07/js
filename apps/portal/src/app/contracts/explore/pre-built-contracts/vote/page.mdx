# Votación

El contrato de **Votación** está diseñado para grupos como los DAOs para votar sobre propuestas.

Para usar el contrato de votación, también necesitas tener un token ERC20 existente, 
como nuestro [contrato de token](/contracts/explore/pre-built-contracts/token) para que actúe como el token de gobernanza.

## Casos de uso y ejemplos

Puedes usar el contrato de votación para:

- Votar para decidir sobre cambios organizacionales
- Votar sobre la gestión y distribución de fondos en un tesoro
- Votar sobre cualquier otra propuesta dentro de una organización, como un DAO

## Crear y configurar el voto

Aprende cómo crear y configurar tu contrato inteligente usando el [dashboard](https://thirdweb.com/team).

### Crear un contrato de votación

Despliega el contrato de votación en cualquiera de nuestras
[redes compatibles](https://blog.thirdweb.com/guides/which-network-should-you-use) usando el botón a continuación.

### Crear propuestas

Para crear una nueva propuesta para que las personas voten, haz clic en **Crear Propuesta**

### Ver propuestas

Puedes ver todas las propuestas activas y pasadas desde la pestaña **Resumen**.

### Votar en propuestas

Si posees el token de gobernanza, puedes votar en las propuestas activas desde la pestaña **Resumen**.
