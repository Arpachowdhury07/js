# Stake ERC1155

Este contrato permite a los usuarios hacer staking de sus NFTs ERC-1155 y obtener tokens ERC-20 como recompensas por el staking. El administrador del contrato debe establecer las direcciones de NFT y de token de recompensa durante el despliegue. Estas no se pueden cambiar más tarde.

Los NFTs apostados se almacenan dentro del contrato y las recompensas se calculan según el tiempo que estos NFTs estuvieron apostados. El administrador debe mantener un saldo adecuado de tokens de recompensa en el contrato de staking. Estas recompensas se transfieren a los stakers al momento de reclamar.

Cálculo de recompensas al momento de reclamar:

`rewardsPerTokenStaked = (stakingDuration * rewardsPerUnitTime) / timeUnit`

Nota:

- Los usuarios deben aprobar sus NFTs a la dirección del contrato de staking antes de poder apostarlos
- El administrador del contrato debe mantener un saldo adecuado de Tokens de Recompensa en el contrato de staking -- usa la función de depósito para depositar los tokens de recompensa
- El tiempo predeterminado y las recompensas se aplicarán para todos los token-Ids, a menos que se establezcan valores específicos para un token
