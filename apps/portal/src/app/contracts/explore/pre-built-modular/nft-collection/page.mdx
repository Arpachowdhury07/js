# Desplegar un contrato modular de Colección de NFT

El contrato de Colección de NFT crea una colección única de NFTs basada en el estándar ERC-721A. Los NFTs se acuñan en la cartera del creador a medida que se agregan a la colección. Los NFTs siguen siendo transferibles, pueden venderse en un marketplace y realizar cualquier otra acción definida por ERC-721A.

Para casos de uso avanzados, el contrato de Colección de NFT incluye capacidades de acuñación basadas en firmas.

### NFT Collection vs NFT Drop

El contrato de Colección de NFT no tiene capacidades de acuñación perezosa ni condiciones de reclamación. Si prefieres definir configuraciones específicas como límites de reclamación, horarios de inicio y fin para reclamar, y más, consulta el contrato de [NFT Drop](https://thirdweb.com/thirdweb.eth/ERC721CoreInitializable?module=eyJwdWJsaXNoZXIiOiJkZXBsb3llci50aGlyZHdlYi5ldGgiLCJtb2R1bGVJZCI6IkNsYWltYWJsZUVSQzcyMSJ9&module=eyJwdWJsaXNoZXIiOiJkZXBsb3llci50aGlyZHdlYi5ldGgiLCJtb2R1bGVJZCI6IkJhdGNoTWV0YXRhdGFFUkM3MjEifQ%3D%3D&module=eyJwdWJsaXNoZXIiOiJkZXBsb3llci50aGlyZHdlYi5ldGgiLCJtb2R1bGVJZCI6IkJhdGNoTWV0YXRhdGFFUkM3MjEifQ%3D%3D&displayName=Modular+NFT+Drop).

### Casos de uso y ejemplos

- Crear una colección 1-de-muchas de fotografías, obras de arte u otros medios para ser coleccionados o vendidos en un marketplace
- Crear un NFT 1-de-1 que proporcione utilidad especial o pueda ser usado para restringir el acceso a una aplicación
- Crear una colección de NFTs hecha por la comunidad usando acuñación basada en firmas

### Módulos instalados por defecto

- Mintable
- BatchMetadata
- Transferible

### Contrato Modular

Este contrato se crea usando el marco de contratos modulares. Los módulos de este contrato son actualizables. [Aprende más sobre Contratos Modulares.](/contracts/modular-contracts/overview)

### Configurar Módulos

Para configurar los módulos, dirígete a la vista de Módulos:

- Configura el módulo MintableERC721 para facilitar la acuñación de tokens ERC721 con configuraciones de ventas.

<DocImage src={collectionMintable} alt="Captura de pantalla del módulo ERC721 Mintable"/>

- Configura BatchMetadataERC721 para cargar los metadatos de los NFTs.

<DocImage src={collectionBatchMetadata} alt="Captura de pantalla del módulo ERC721 BatchMetadata"/>

- Configura el módulo Transferable721 para restringir las transferencias, si lo deseas.

<DocImage src={collectionTransferable} alt="Captura de pantalla del módulo ERC721 Transferable"/>
