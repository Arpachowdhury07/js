# Desplegar un contrato modular de Token

El contrato modular de Token crea monedas digitales y es compatible con el estándar ERC-20. Este contrato define un conjunto común de reglas para los tokens, incluyendo cómo pueden ser transferidos, cómo se aprueban las transacciones y cómo se accede a los datos sobre los saldos de los tokens.

### Token vs Token Drop

El contrato de Token no permite a los usuarios reclamar tokens bajo los criterios de las condiciones de reclamación. Si prefieres establecer límites como cuándo y cómo tus usuarios pueden reclamar tokens definidos a través de listas de permisos, fechas de lanzamiento y límites de reclamación, consulta el contrato de [Token Drop](https://thirdweb.com/thirdweb.eth/ERC20CoreInitializable?module=eyJwdWJsaXNoZXIiOiJkZXBsb3llci50aGlyZHdlYi5ldGgiLCJtb2R1bGVJZCI6IkNsYWltYWJsZUVSQzIwIn0%3D&module=eyJwdWJsaXNoZXIiOiJkZXBsb3llci50aGlyZHdlYi5ldGgiLCJtb2R1bGVJZCI6IlRyYW5zZmVyYWJsZUVSQzIwIn0%3D&displayName=Modular+Token+Drop).

### Casos de uso

- Vende NFTs en un marketplace usando un token personalizado
- Crea criptomonedas personalizadas usadas como moneda digital, para pagos, intercambios, stablecoins, ICOs, y más
- Recompensa a los usuarios con tokens por acciones específicas completadas en juegos u otras aplicaciones
- Otorga derechos de gobernanza usando tokens para votar sobre actualizaciones de protocolos, decisiones, asignaciones de recursos del tesoro

### Módulos instalados por defecto

- Transferible
- Mintable

### Contrato Modular

Este contrato se crea usando el marco de contratos modulares. Los módulos de este contrato son actualizables. [Aprende más sobre Contratos Modulares.](https://portal.thirdweb.com/contracts/modular-contracts/overview)

### Configurar Módulos

Para configurar los módulos, dirígete a la vista de Módulos:

- Configura el Módulo Transferible para restringir las transferencias, si lo deseas.
