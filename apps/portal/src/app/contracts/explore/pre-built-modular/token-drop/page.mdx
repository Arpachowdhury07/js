# Desplegar un contrato modular de Token Drop

El contrato de Token Drop crea monedas digitales que se liberan a un precio determinado y es compatible con el estándar [ERC-20](https://eips.ethereum.org/EIPS/eip-20). Este contrato define un conjunto común de reglas para los tokens, incluyendo cómo pueden ser transferidos, cómo se aprueban las transacciones y cómo se accede a los datos sobre los saldos de los tokens.

Token Drop incluye configuraciones llamadas condiciones de reclamación que permiten a los desplegadores establecer precios para los tokens por fase de reclamación y límites sobre cuántos tokens se liberarán. Estas se habilitan a través del módulo Claimable.

### Token vs Token Drop

El contrato de Token Drop permite a los usuarios reclamar tokens según configuraciones específicas establecidas mediante las Condiciones de Reclamación. Si no necesitas establecer límites o configuraciones especiales sobre cómo reclamar, consulta el contrato simple de [Token](https://thirdweb.com/thirdweb.eth/ERC20CoreInitializable?module=eyJwdWJsaXNoZXIiOiJkZXBsb3llci50aGlyZHdlYi5ldGgiLCJtb2R1bGVJZCI6Ik1pbnRhYmxlRVJDMjAifQ%3D%3D&module=eyJwdWJsaXNoZXIiOiJkZXBsb3llci50aGlyZHdlYi5ldGgiLCJtb2R1bGVJZCI6IlRyYW5zZmVyYWJsZUVSQzIwIn0%3D&displayName=Modular+Token).

### Casos de uso

- Liberar criptomonedas a un precio determinado. Ejemplo: 1 MATIC por token
- Permitir que carteras específicas reclamen tokens antes de los lanzamientos públicos
- Permitir a los usuarios reclamar tokens hasta una fecha límite establecida

### Módulos instalados por defecto

- Transferible
- Reclamable

### Contrato Modular

Este contrato se crea usando el marco de contratos modulares. Los módulos de este contrato son actualizables. [Aprende más sobre Contratos Modulares.](/contracts/modular-contracts/overview)

### Configurar Módulos

Para configurar los módulos, dirígete a la vista de Módulos:

- Configura el Módulo Transferible para restringir las transferencias, si lo deseas.

<DocImage src={tokenDropTransferable} alt="Captura de pantalla del módulo ERC20 Transferible"/>

- Configura el módulo Reclamable para acuñar tokens, establecer condiciones de reclamación y actualizar el destinatario de la venta principal.

<DocImage src={tokenDropClaimable} alt="Captura de pantalla del módulo ERC20 Reclamable"/>

<Callout variant='info' title="Instalar Módulos">
