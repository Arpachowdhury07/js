# Desplegar un contrato modular de NFT Drop

El contrato de NFT Drop crea una colección única de NFTs basada en el estándar [ERC-721A](https://www.erc721a.org/). NFT Drop te permite definir condiciones de reclamación o configuraciones que definen cómo un usuario puede reclamar un NFT.

### NFT Drop vs NFT Collection

NFT Drop tiene el módulo Claimable que habilita las condiciones de reclamación. Si prefieres no definir configuraciones específicas como límites de reclamación, horarios de inicio y fin para reclamar, y más, consulta el contrato de [NFT Collection](https://thirdweb.com/thirdweb.eth/ERC721CoreInitializable?module=eyJwdWJsaXNoZXIiOiJkZXBsb3llci50aGlyZHdlYi5ldGgiLCJtb2R1bGVJZCI6Ik1pbnRhYmxlRVJDNzIxIn0%3D&module=eyJwdWJsaXNoZXIiOiJkZXBsb3llci50aGlyZHdlYi5ldGgiLCJtb2R1bGVJZCI6IkJhdGNoTWV0YXRhdGFFUkM3MjEifQ%3D%3D&module=eyJwdWJsaXNoZXIiOiJkZXBsb3llci50aGlyZHdlYi5ldGgiLCJtb2R1bGVJZCI6IlRyYW5zZmVyYWJsZUVSQzcyMSJ9&displayName=Modular+NFT+Collection).

### Casos de uso y ejemplos

- Crear una colección de fotos, arte u otros medios donde cada NFT tenga una combinación diferente de características
- Liberar NFTs en fases con exclusividad, descuentos o fases públicas
- Crear NFT Drops con acceso restringido para que las carteras permitidas reclamen

### Módulos instalados por defecto

- Reclamable
- BatchMetadata
- Royalty

Dirígete a Explorar, localiza el [contrato modular de NFT Drop](https://thirdweb.com/thirdweb.eth/ERC721CoreInitializable?module=eyJwdWJsaXNoZXIiOiJkZXBsb3llci50aGlyZHdlYi5ldGgiLCJtb2R1bGVJZCI6IkNsYWltYWJsZUVSQzcyMSJ9&module=eyJwdWJsaXNoZXIiOiJkZXBsb3llci50aGlyZHdlYi5ldGgiLCJtb2R1bGVJZCI6IkJhdGNoTWV0YWRhdGFFUkM3MjEifQ%3D%3D&module=eyJwdWJsaXNoZXIiOiJkZXBsb3llci50aGlyZHdlYi5ldGgiLCJtb2R1bGVJZCI6IlJveWFsdHlFUkM3MjEifQ%3D%3D&displayName=Modular+NFT+Drop), y selecciona el botón Desplegar ahora.

<DocImage src={nftDropStep1} alt="Captura de pantalla del contrato de NFT Drop" />

</Step>

<Step title="Configurar los metadatos del contrato">
Rellena los metadatos del contrato, incluyendo Nombre, Símbolo, Descripción e Imagen.

Los metadatos del contrato sirven como tu identificador único.

<DocImage src={nftDropStep2} alt="Captura de pantalla de la configuración de los metadatos de NFT Drop" />

</Step>

<Step title="Seleccionar la cadena">
Selecciona la cadena preferida en la que deseas desplegar y selecciona Desplegar ahora.

<DocImage src={nftDropStep3} alt="Captura de pantalla de la selección de la cadena para el contrato de NFT Drop" />

<Callout variant="info" title="Cualquier red compatible con EVM">
	El contrato de NFT Drop es desplegable en cualquier red compatible con EVM. [Ver todas las redes disponibles.](https://thirdweb.com/chainlist)
</Callout>
Si has iniciado sesión con una cartera EOA, se te pedirá que firmes la transacción. Después de eso, tu contrato será desplegado y podrás comenzar a configurar los módulos.
</Step>

</Steps>
