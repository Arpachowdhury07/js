# ERC20DropVote

El contrato base `ERC20Drop` agrega la extensión [Drop Single Phase](/contracts/build/extensions/general/DropSinglePhase) al contrato [ERC20 base](/contracts/build/base-contracts/erc-20/base)
para permitirte distribuir tus tokens bajo los criterios de condiciones de reclamo. También
implementa el contrato [ERC20Votes](https://github.com/OpenZeppelin/openzeppelin-contracts/blob/master/contracts/token/ERC20/extensions/ERC20Votes.sol),
que proporciona funcionalidades de votación y delegación, junto con delegación por firma.


```solidity
import "@thirdweb-dev/contracts/base/ERC20DropVote.sol";
```

<GithubButtonLink href="https://github.com/thirdweb-dev/contracts/blob/main/contracts/base/ERC20DropVote.sol" />

## Uso

Importa el contrato e inhíerelo.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@thirdweb-dev/contracts/base/ERC20DropVote.sol";

contract MyToken is ERC20DropVote {
	  constructor(
        address _defaultAdmin,
        string memory _name,
        string memory _symbol,
        address _primarySaleRecipient
    )
        ERC20DropVote(
            _defaultAdmin,
            _name,
            _symbol,
            _primarySaleRecipient
        )
    {}
}
```

```solidity
/// @dev Recoge y distribuye el valor de la venta primaria de los tokens que se están reclamando.
function _collectPriceOnClaim(
    address _primarySaleRecipient,
    uint256 _quantityToClaim,
    address _currency,
    uint256 _pricePerToken
) internal virtual override {
    if (_pricePerToken == 0) {
        return;
    }

    uint256 totalPrice = (_quantityToClaim * _pricePerToken) / 1 ether;
    require(totalPrice > 0, "quantity too low");

    if (_currency == CurrencyTransferLib.NATIVE_TOKEN) {
        require(msg.value == totalPrice, "Must send total price.");
    }

    address saleRecipient = _primarySaleRecipient == address(0) ? primarySaleRecipient() : _primarySaleRecipient;
    CurrencyTransferLib.transferCurrency(_currency, msg.sender, saleRecipient, totalPrice);
}
```
