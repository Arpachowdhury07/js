# Build (Solidity SDK)

**Build** o **Solidity SDK** proporciona las herramientas necesarias para construir contratos inteligentes personalizados de manera más eficiente, ofreciendo un conjunto de contratos base preconstruidos y componentes reutilizables, mejor conocidos como extensiones.

<DocImage src={extensinoModelImage} />

#### Características

- **Soporte para EIPs populares:** Integra fácilmente características y funcionalidades comunes de contratos inteligentes de EIPs populares como ERC-20, ERC-721, ERC-1155, permisos, metadatos y más.
- **Código abierto:** El Solidity SDK es de código abierto y, por lo tanto, está disponible para que cualquier usuario pueda auditarlo o contribuir al repositorio. [Ver el repositorio de contratos.](https://github.com/thirdweb-dev/contracts)
- **Contratos base optimizados y auditados:** Usar contratos base proporciona una base optimizada para gas y auditada para construir tus proyectos.
- **Herramientas de gestión y construcción desbloqueadas:** Cada extensión heredada desbloquea funciones fáciles de usar en los SDKs de interacción con contratos, paneles de administración personalizados y feeds de datos adaptados.
