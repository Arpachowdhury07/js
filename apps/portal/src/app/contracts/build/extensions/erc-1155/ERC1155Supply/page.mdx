Ver todos los NFTs en el contrato implementando la función `totalSupply` en tu contrato ERC1155.

## Uso

Este es un contrato inteligente de ejemplo que demuestra cómo heredar de esta extensión y sobrecargar las funciones para agregar (opcionalmente) funcionalidad personalizada.

El contrato [ERC1155Drop](/contracts/build/base-contracts/erc-1155/drop) viene con una función `totalSupply`
que devuelve el número total de NFTs que se han acuñado en la colección (incluyendo los tokens que han sido quemados).

Esto significa que ya puedes ver el suministro total de NFTs en la colección al implementar el contrato `ERC1155Base`.

Puedes sobrecargar opcionalmente la funcionalidad de esta función como se detalla a continuación:

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@thirdweb-dev/contracts/base-contracts/ERC71155Base.sol";

contract Contract is ERC1155Base {
    constructor(
        string memory _name,
        string memory _symbol
    )
        ERC1155Base(
            _name,
            _symbol
        )
    {}

    function totalSupply() public view override returns (uint256) {
        // Your custom implementation here (overriding this function is optional)
    }

}
```
