# ERC20BatchMintable

```solidity
import "@thirdweb-dev/eip/interface/IERC20.sol"
import "@thirdweb-dev/contracts/extension/multicall.sol";
import "@thirdweb-dev/contracts/extension/interface/IMintableERC20.sol";
```

Habilita la acuñación de múltiples NFTs a la vez en una sola transacción implementando las extensiones [`ERC20`](/contracts/build/extensions/erc-20/ERC20), [`IMintableERC20`](/contracts/build/extensions/erc-20/ERC20BatchMintable), y [`Multicall`](/contracts/build/extensions/general/Multicall).

Esta es una extensión que es detectable en el panel si el contrato inteligente implementa las extensiones
[`ERC20`](/contracts/build/extensions/erc-20/ERC20), [`Multicall`](/contracts/build/extensions/general/Multicall) y [`ERC20Mintable`](/contracts/build/extensions/erc-20/ERC20BatchMintable).

## Uso

Este es un contrato inteligente de ejemplo que tendrá `ERC20BatchMintable` detectado en el panel. También muestra la función que se puede sobrescribir para agregar lógica personalizada.

```solidity

import "@thirdweb-dev/contracts/extension/Multicall.sol";

    constructor(
        string memory _name,
        string memory _symbol
    )
        ERC20(
            _name,
            _symbol
        )
    {}

    function mintTo(address to, uint256 amount) external override returns (uint256) {
        // Your custom implementation here
    }

}
```
