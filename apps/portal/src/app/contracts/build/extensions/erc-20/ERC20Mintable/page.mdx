# ERC20Mintable

```solidity
import "@thirdweb-dev/contracts/extension/interface/IMintableERC20.sol";
```

Habilita la acuñación de nuevos tokens implementando la interfaz `IMintableERC20`.

<Callout title="info" variant="info">

Esta extensión es una interfaz y requiere que todas las funciones sean implementadas.

</Callout>

<br />

<GithubButtonLink href="https://github.com/thirdweb-dev/contracts/blob/main/contracts/extension/interface/IMintableERC20.sol" />

## Uso

Este es un contrato inteligente de ejemplo que demuestra cómo heredar de esta extensión y sobrescribir las funciones para agregar (de manera opcional) funcionalidad personalizada.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@thirdweb-dev/contracts/openzeppelin-presets/token/ERC20/ERC20.sol";
import "@thirdweb-dev/contracts/extension/interface/IMintableERC20.sol";

contract Contract is ERC20, IMintableERC20 {
    constructor(
        string memory _name,
        string memory _symbol
    )
        ERC20(
            _name,
            _symbol
        )
    {}

    function mintTo(address to, uint256 amount) external override {
        // Tu implementación personalizada aquí
    }
}
```
