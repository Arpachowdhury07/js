# ERC20ClaimConditions

```solidity
import "@thirdweb-dev/contracts/eip/interface/IERC20.sol";
import "@thirdweb-dev/contracts/extension/DropSinglePhase.sol";
```

Permite que otras carteras reclamen/mintan tokens desde el contrato bajo los criterios de las condiciones de reclamo implementando las extensiones [`ERC20`](/contracts/build/extensions/erc-20/ERC20) y [`DropSinglePhase`](/contracts/build/extensions/general/DropSinglePhase).

Esta es una extensión que es detectable en el panel si el contrato inteligente implementa las extensiones [`ERC20`](/contracts/build/extensions/erc-20/ERC20) y [`DropSinglePhase`](/contracts/build/extensions/general/DropSinglePhase).
