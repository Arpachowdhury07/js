# ERC20

```solidity
import "@thirdweb-dev/contracts/eip/interface/IERC20.sol";
```
ERC20 es el estándar para representar tokens fungibles; donde cada token tiene el mismo valor y es intercambiable.

<Callout title="info" variant="info">
  Esta extensión es una interfaz y requiere que todas las funciones sean
  implementadas
</Callout>

<br />

<GithubButtonLink href="https://github.com/thirdweb-dev/contracts/blob/main/contracts/eip/interface/IERC20.sol" />

## Uso

Este es un contrato inteligente de ejemplo que demuestra cómo heredar de esta extensión
y sobrescribir las funciones para agregar (opcionalmente) funcionalidades personalizadas.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@thirdweb-dev/contracts/eip/interface/IERC20.sol";

contract Contract is IERC20 {
    constructor(
        string memory _name,
        string memory _symbol
    ) {}

    function totalSupply() external view returns (uint256) {
      // Tu implementación personalizada aquí
    }

    function balanceOf(address who) external view returns (uint256) {
      // Tu implementación personalizada aquí
    }

    function allowance(address owner, address spender) external view returns (uint256) {
      // Tu implementación personalizada aquí
    }

    function transfer(address to, uint256 value) external returns (bool) {
      // Tu implementación personalizada aquí
    }

    function approve(address spender, uint256 value) external returns (bool) {
      // Tu implementación personalizada aquí
    }

    function transferFrom(
        address from,
        address to,
        uint256 value
    ) external returns (bool); {
      // Tu implementación personalizada aquí
    }
}
```
