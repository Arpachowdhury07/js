# Ownable

```solidity
import "@thirdweb-dev/contracts/extension/Ownable.sol";
```

La extensión del contrato inteligente `Ownable` es utilizable con cualquier contrato inteligente base. Te permite establecer un propietario para tu contrato inteligente.

<GithubButtonLink href="https://github.com/thirdweb-dev/contracts/blob/main/contracts/extension/Ownable.sol" />

## Uso

La extensión `Ownable` es un _contrato abstracto_, y espera que implementes las siguientes funciones por ti mismo:

| Nombre                                                                      | Tipo                    | Retorna | Descripción                                                                                                                  |
| --------------------------------------------------------------------------- | ----------------------- | ------- | ---------------------------------------------------------------------------------------------------------------------------- |
| [`_canSetOwner`](/contracts/build/extensions/general/Ownable#_cansetowner)  | internal view _virtual_ | `bool`  | Se ejecuta en cada intento de establecer el propietario del contrato. Devuelve si el propietario puede ser establecido en el contexto de ejecución dado. |

Este es un contrato inteligente de ejemplo que demuestra cómo heredar de esta extensión y sobrescribir las funciones para agregar (opcionalmente) funcionalidad personalizada.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@thirdweb-dev/contracts/extension/Ownable.sol";

contract MyContract is Ownable {
    /**
     *  Esta función devuelve quién está autorizado para establecer el propietario de tu contrato.
     *
     *  Como EJEMPLO, solo permitiremos que el propietario actual establezca el nuevo propietario del contrato.
     *
     *  DEBES completar el cuerpo de esta función para usar la extensión `Ownable`.
     */
    function _canSetOwner() internal virtual view override returns (bool) {
        return msg.sender == owner();
    }
}
```

## Contratos Base que Implementan Esta Extensión

Todos los contratos base implementan esta extensión.

## Referencia Completa de la API

<Details summary="owner">

```solidity
function owner() public view override returns (address);
```

- Devuelve el propietario del contrato.
- Establece este valor usando la función `setOwner`.

</Details>

<Details summary="setOwner">

```solidity
function setOwner(address _newOwner) external;
```

- Permite que una billetera autorizada establezca un nuevo propietario para el contrato. Este valor puede ser leído desde la función `owner`.
- Parámetro `_newOwner`: La dirección para establecer como propietario del contrato.
- La función `_canSetOwner` se ejecuta en cada llamada a esta función. Si el valor devuelto de `_canSetOwner` es `false`, la llamada a `setOwner` se revertirá.

</Details>

<Details summary="_setupOwner">

```solidity
function _setupOwner(address _newOwner) internal;
```

- Establece el propietario del contrato inteligente. Este valor puede ser leído desde la función `owner`.
- Parámetro `_newOwner`: La dirección para establecer como propietario del contrato.
- La función `_canSetOwner` _no_ se ejecuta en una llamada a esta función.

</Details>

<Details summary="_canSetOwner">

```solidity
function _canSetOwner() internal view virtual returns (bool);
```

- Se ejecuta en cada llamada a la función `setOwner`.
- Devuelve si un nuevo propietario puede ser establecido en el contexto de ejecución dado.
- Por ejemplo, esta función puede verificar si la billetera que llama a `setOwner` es el propietario del contrato, y hacer cumplir que solo el propietario actual pueda llamar exitosamente a `setOwner`.

</Details>
