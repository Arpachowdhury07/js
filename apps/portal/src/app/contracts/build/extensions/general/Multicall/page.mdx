# Multicall

```solidity
import "@thirdweb-dev/contracts/extension/Multicall.sol";
```

El contrato inteligente `Multicall` es una extensión utilizable con cualquier contrato inteligente base. Proporciona una función `multicall` para agrupar múltiples llamadas en una sola llamada externa.

<Callout title="info" variant="info">

Se recomienda usar esta extensión junto con la extensión [`ERC721Mintable`](/contracts/build/extensions/erc-721/ERC721Mintable), [`ERC1155Mintable`](/contracts/build/extensions/erc-1155/ERC1155Mintable) o [`ERC20Mintable`](/contracts/build/extensions/erc-20/ERC20BatchMintable) para desbloquear la detección de la extensión del panel de control [`ERC721BatchMintable`](/contracts/build/extensions/erc-721/ERC721BatchMintable), [`ERC1155BatchMintable`](/contracts/build/extensions/erc-1155/ERC1155BatchMintable) o [`ERC20BatchMintable`](/contracts/build/extensions/erc-20/ERC20BatchMintable).

</Callout>

## Uso

Este es un contrato inteligente de ejemplo que demuestra cómo heredar de esta extensión y sobrescribir las funciones para agregar (opcionalmente) funcionalidad personalizada.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@thirdweb-dev/contracts/extension/Multicall.sol";

contract MyContract is Multicall { ... }
```

## Referencia Completa de la API

<Details summary="multicall">

```solidity
function multicall(bytes[] calldata data) external virtual override returns (bytes[] memory results);
```

- Recibe y ejecuta un lote de llamadas a funciones en el contrato.
- Parámetro `data`: Los datos en bytes que conforman el lote de llamadas a funciones a ejecutar.
- Valor de retorno `results`: Los datos en bytes que conforman el resultado del lote de llamadas a funciones ejecutadas.

</Details>
