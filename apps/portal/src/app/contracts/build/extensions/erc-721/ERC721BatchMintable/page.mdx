# ERC721BatchMintable

```solidity
import "@thirdweb-dev/eip/ERC721.sol"
import "@thirdweb-dev/contracts/extension/multicall.sol";
import "@thirdweb-dev/contracts/extension/interface/IMintableERC721.sol";
```

Habilita la acuñación de múltiples NFTs a la vez en una sola transacción mediante la implementación de las extensiones [`ERC721`](/contracts/build/extensions/erc-721/ERC721), [`IMintableERC721`](/contracts/build/extensions/erc-721/ERC721Mintable) y [`Multicall`](/contracts/build/extensions/general/Multicall).

Esta es una extensión que es detectable en el panel de control si el contrato inteligente implementa las extensiones
[ERC721](/contracts/build/extensions/erc-721/ERC721), [Multicall](/contracts/build/extensions/general/Multicall) y [ERC721Mintable](/contracts/build/extensions/erc-721/ERC721Mintable).

## Uso

Este es un ejemplo de contrato inteligente que tendrá detectado ERC721BatchMintable en el panel de control. También muestra las funciones que se pueden (opcionalmente) sobrescribir para agregar lógica personalizada.
