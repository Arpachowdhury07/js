# ERC721ClaimConditions 

```solidity
import "@thirdweb-dev/contracts/eip/ERC721.sol";
import "@thirdweb-dev/contracts/extension/DropSinglePhase.sol";
```

Permite que otras billeteras reclamen/minte tokens del contrato bajo los criterios de condiciones de reclamo implementando las extensiones [`ERC721`](/contracts/build/extensions/erc-721/ERC721) y [`DropSinglePhase`](/contracts/build/extensions/general/DropSinglePhase).

Esta es una extensión que se puede detectar en el panel si el contrato inteligente implementa las extensiones [`ERC721`](/contracts/build/extensions/erc-721/ERC721) y [`DropSinglePhase`](/contracts/build/extensions/general/DropSinglePhase).
