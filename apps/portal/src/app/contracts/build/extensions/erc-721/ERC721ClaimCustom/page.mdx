# ERC721ClaimCustom

```solidity
import "@thirdweb-dev/contracts/eip/ERC721.sol";
import "@thirdweb-dev/contracts/extension/interface/IClaimableERC721.sol";
```

Permite que otras billeteras reclamen/minten tokens que has mintado de manera perezosa al
implementar las extensiones [`ERC721`](/contracts/build/extensions/erc-721/ERC721) y [`ERC721Claimable`](/contracts/build/extensions/erc-721/ERC721Claimable).

Esta es una extensión que se puede detectar en el panel si el contrato inteligente implementa las
extensiones [`ERC721`](/contracts/build/extensions/erc-721/ERC721) y [`ERC721Claimable`](/contracts/build/extensions/erc-721/ERC721Claimable).
