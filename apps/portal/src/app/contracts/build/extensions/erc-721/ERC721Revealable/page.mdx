# ERC721Revealable

```solidity
import "@thirdweb-dev/contracts/eip/ERC721.sol";
import "@thirdweb-dev/contracts/extension/DelayedReveal.sol";
```

Crea lotes de NFTs de ['revelación retrasada'](/glossary/delayed-reveal) que pueden ser revelados en una fecha posterior, después de haber sido acuñados y reclamados.

Esta es una extensión que se puede detectar en el panel si el contrato inteligente implementa las extensiones
[`ERC721`](/contracts/build/extensions/erc-721/ERC721) y [`DelayedReveal`](/contracts/build/extensions/general/DelayedReveal).

<GithubButtonLink href="https://github.com/thirdweb-dev/contracts/blob/main/contracts/extension/DelayedReveal.sol" />

## Uso

Este es un contrato inteligente de ejemplo que demuestra cómo heredar de esta extensión y sobrescribir las funciones para agregar (opcionalmente) funcionalidad personalizada.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@thirdweb-dev/contracts/eip/ERC721A.sol";
import "@thirdweb-dev/contracts/extension/LazyMint.sol";
import "@thirdweb-dev/contracts/extension/DelayedReveal.sol";

contract Contract is ERC721A, LazyMint, DelayedReveal {
    constructor(
        string memory _name,
        string memory _symbol
    )
        ERC721A(
            _name,
            _symbol
        )
    {}
  function reveal(uint256 _index, bytes calldata _key)
        external
        override
        returns (string memory revealedURI)
    {
      // Your custom implementation here
    }

    function _canLazyMint() internal view override returns (bool) {
        // Your custom implementation here
    }
}
```
