# ERC-20 Core

## Funciones de Callback Soportadas

| Función                          | Modo      |
|-----------------------------------|-----------|
| `beforeMintERC20`                 | REQUERIDO |
| `beforeMintWithSignatureERC20`    | REQUERIDO |
| `beforeTransferERC20`             | OPCIONAL |
| `beforeBurnERC20`                 | OPCIONAL |
| `beforeApproveERC20`              | OPCIONAL |

## Interfaces Soportadas

| Interfaz        | ID          |
|------------------|-------------|
| `ERC165`         | `0x01ffc9a7`|
| `ERC-7572`       | `0xe8a3d485`|
| `ERC-173`        | `0x7f5828d0`|
| `IERC20`         | `0x36372b07`|

## Funciones Soportadas

| Función                | Descripción                                                                 |
|-------------------------|-----------------------------------------------------------------------------|
| `_initialize`           | Inicializa el contrato con parámetros para el nombre, símbolo, URI de metadatos, propietario, módulos y sus datos. |
| `contractURI`           | Devuelve la URI de metadatos del contrato.                                  |
| `name`                  | Devuelve el nombre del token.                                               |
| `symbol`                | Devuelve el símbolo del token.                                             |
| `supportsInterface`     | Verifica si el contrato implementa una interfaz con el ID de interfaz dado. |
| `getSupportedCallbackFunctions` | Devuelve las funciones de callback soportadas.                  |
| `setContractURI`        | Establece la URI de metadatos del contrato. Solo puede ser llamada por el administrador del contrato. |
| `mint`                  | Mina tokens a una dirección especificada, llamando a la función beforeMint. |
| `mintWithSignature`     | Mina tokens con una firma, llamando a la función beforeMintWithSignature.  |
| `burn`                  | Quema tokens de una dirección especificada, llamando a la función beforeBurn si existe.|
| `transfer`              | Transfiere tokens a un destinatario, llamando a la función beforeTransfer si existe.|
| `transferFrom`          | Transfiere tokens de un remitente a un destinatario, llamando a la función beforeTransfer si existe.|
| `approve`               | Aprueba un gastador para gastar tokens en nombre de un propietario, llamando a la función beforeApprove si existe.|
| `permit`                | Establece la asignación basada en la aprobación firmada del propietario del token, llamando a la función beforeApprove. |
