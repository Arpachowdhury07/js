# ERC-721 Core

## Funciones de Callback Soportadas

| Función                          | Modo      |
|-----------------------------------|-----------|
| `beforeMintERC721`                | REQUERIDO |
| `beforeMintWithSignatureERC721`   | REQUERIDO |
| `beforeTransferERC721`            | OPCIONAL |
| `beforeBurnERC721`                | OPCIONAL |
| `beforeApproveERC721`             | OPCIONAL |
| `beforeApproveForAll`             | OPCIONAL |
| `onTokenURI`                      | REQUERIDO |
| `updateMetadataERC721`            | REQUERIDO |

## Funciones Soportadas

| Función                | Descripción                                                                 |
|-------------------------|-----------------------------------------------------------------------------|
| `_initialize`           | Inicializa el contrato con parámetros para el nombre, símbolo, URI de metadatos, propietario, módulos y sus datos. |
| `contractURI`           | Devuelve la URI de metadatos del contrato.                                  |
| `startTokenId`          | Devuelve el ID inicial del token para las acuñaciones secuenciales.         |
| `totalMinted`           | Devuelve el número total de tokens acuñados.                                |
| `tokenURI`              | Devuelve los metadatos del token de un NFT.                                 |
| `supportsInterface`     | Verifica si el contrato implementa una interfaz con el ID de interfaz dado. |
| `getSupportedCallbackFunctions` | Devuelve las funciones de callback soportadas.                  |
| `setContractURI`        | Establece la URI de metadatos del contrato. Solo puede ser llamada por el administrador del contrato. |
| `mint`                  | Mina tokens a una dirección especificada, llamando a la función beforeMint. |
| `mintWithSignature`     | Mina tokens con una firma, llamando a la función beforeMintWithSignature.  |
| `burn`                  | Quema un token especificado, llamando a la función beforeBurn si existe.   |
| `transferFrom`          | Transfiere la propiedad de un NFT, llamando a la función beforeTransfer si existe. |
| `approve`               | Aprueba una dirección para transferir un NFT específico, llamando a la función beforeApprove si existe. |
| `setApprovalForAll`     | Aprueba o revoca la aprobación de un operador para transferir o aprobar todos los NFTs del llamador. |
