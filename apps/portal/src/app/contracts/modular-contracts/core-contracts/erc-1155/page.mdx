# ERC-1155 Core

## Funciones de Callback Soportadas

| Función                          | Modo      |
|-----------------------------------|-----------|
| `beforeMintERC1155`               | REQUERIDO  |
| `beforeMintWithSignatureERC1155`  | REQUERIDO  |
| `beforeTransferERC1155`           | OPCIONAL  |
| `beforeBatchTransferERC1155`      | OPCIONAL  |
| `beforeBurnERC1155`               | OPCIONAL  |
| `beforeApproveForAll`             | OPCIONAL  |
| `onTokenURI`                      | REQUERIDO  |
| `updateMetadataERC1155`           | REQUERIDO  |
| `updateTokenIdERC1155`            | OPCIONAL  |

## Funciones Soportadas

| Función                | Descripción                                                                 |
|-------------------------|-----------------------------------------------------------------------------|
| `_initialize`           | Inicializa el contrato con parámetros para nombre, símbolo, URI de metadatos, propietario, módulos y sus datos. |
| `contractURI`           | Devuelve la URI de metadatos del contrato.                                  |
| `totalSupply`           | Devuelve el suministro total de un tokenId especificado.                    |
| `uri`                   | Devuelve los metadatos del token de un NFT.                                 |
| `supportsInterface`     | Verifica si el contrato implementa una interfaz con el ID de interfaz dado. |
| `getSupportedCallbackFunctions` | Devuelve las funciones de callback soportadas.                  |
| `setContractURI`        | Establece la URI de metadatos del contrato. Solo puede ser llamada por el administrador del contrato. |
| `mint`                  | Mina tokens a una dirección especificada, llamando a la función beforeMint. |
| `mintWithSignature`     | Mina tokens con una firma, llamando a la función beforeMintWithSignature.  |
| `burn`                  | Quema un token especificado, llamando a la función beforeBurn si existe.   |
| `safeTransferFrom`      | Transfiere la propiedad de un NFT, llamando a la función beforeTransfer si existe. |
| `safeBatchTransferFrom` | Realiza transferencias masivas de propiedad de NFTs, llamando a la función beforeBatchTransfer si existe. |
| `setApprovalForAll`     | Aprueba o revoca la aprobación de un operador para transferir o aprobar todos los NFTs del llamador, llamando a la función beforeApproveForAll si existe. |
