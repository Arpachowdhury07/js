# OpenEditionMetadataERC721

## Descripción

El contrato `OpenEditionMetadataERC721` es un módulo que proporciona gestión de metadatos para tokens ERC-721 de edición abierta. Incluye funcionalidades para configurar y recuperar metadatos compartidos para los tokens, haciendo la gestión de metadatos simple y eficiente.

<GithubButtonLink href="https://github.com/thirdweb-dev/modular-contracts/blob/dev/src/module/token/metadata/OpenEditionMetadataERC721.sol" />

## Funciones de Callback

| Función           | Descripción                                                  |
|-------------------|--------------------------------------------------------------|
| `onTokenURI`      | Función de callback que retorna la URI del token para los tokens ERC-721. |

## Funciones de Fallback

| Función                | Descripción                                                                               |
|------------------------|-------------------------------------------------------------------------------------------|
| `setSharedMetadata`     | Establece los metadatos compartidos para la colección de tokens ERC-721. Requiere el rol de minter. |
