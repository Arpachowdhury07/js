# MintableERC721

## Descripción

El contrato `MintableERC721` es un módulo que proporciona funcionalidades de minting para tokens ERC-721. Este módulo incluye características como minting con y sin firmas, y configuración para los destinatarios de ventas primarias.

<GithubButtonLink href="https://github.com/thirdweb-dev/modular-contracts/blob/dev/src/module/token/minting/MintableERC721.sol" />

## Funciones de Callback

| Función                          | Descripción                                                                                                    |
|-----------------------------------|----------------------------------------------------------------------------------------------------------------|
| `beforeMintERC721`                | Función de callback que verifica la autorización de minting antes de hacer minting de tokens ERC-721.         |
| `beforeMintWithSignatureERC721`   | Función de callback que verifica una solicitud de minting con una firma y procesa el minting de acuerdo con ello. |

## Funciones de Fallback

| Función                 | Descripción                                                              |
|--------------------------|--------------------------------------------------------------------------|
| `getSaleConfig`          | Retorna la configuración de venta para un token, incluyendo el destinatario de la venta primaria. |
| `setSaleConfig`          | Establece la configuración de venta para un token, incluyendo el destinatario de la venta primaria. Requiere el rol de administrador.       |

## Interfaces Requeridas

| Interface  | ID          |
|------------|-------------|
| `ERC-721`  | `0x80ac58cd`|

## Callback de Instalación

Este módulo utiliza un callback de instalación para configurar los parámetros iniciales, como el destinatario de la venta primaria.
