# ERC-721 Transferible

## Descripción

El contrato `TransferableERC721` es un módulo que proporciona funcionalidad de control de transferencias para tokens ERC-721. Este contrato permite habilitar o deshabilitar las transferencias de tokens globalmente o para direcciones específicas, añadiendo una capa adicional de control sobre cómo se pueden mover los tokens.

<GithubButtonLink href="https://github.com/thirdweb-dev/modular-contracts/blob/dev/src/module/token/transferable/TransferableERC721.sol" />

## Funciones de Callback

| Función             | Descripción                                                                |
|---------------------|----------------------------------------------------------------------------|
| `beforeTransferERC721`| Función de callback para `ERC721.transferFrom`/`safeTransferFrom`. Asegura que las transferencias estén habilitadas antes de permitirlas. |

## Funciones de Fallback

| Función                | Descripción                                                                             |
|------------------------|-----------------------------------------------------------------------------------------|
| `isTransferEnabled`     | Devuelve si las transferencias están habilitadas globalmente para el token.             |
| `isTransferEnabledFor`  | Devuelve si las transferencias están habilitadas para una dirección específica.          |
| `setTransferable`       | Establece el estado de transferibilidad para el token globalmente. Requiere el rol de administrador. |
| `setTransferableFor`    | Establece el estado de transferibilidad para una dirección específica. Requiere el rol de administrador. |
