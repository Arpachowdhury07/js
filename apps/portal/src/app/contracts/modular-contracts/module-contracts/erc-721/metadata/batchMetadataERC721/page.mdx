# BatchMetadataERC721

## Descripción

El contrato `BatchMetadataERC721` es un módulo que proporciona gestión de metadatos en lotes para tokens ERC-721. Incluye funcionalidades para cargar, actualizar y recuperar metadatos de manera eficiente en lotes. Este contrato ayuda a gestionar los metadatos para colecciones grandes de tokens organizándolos en lotes.

<GithubButtonLink href="https://github.com/thirdweb-dev/modular-contracts/blob/dev/src/module/token/metadata/BatchMetadataERC721.sol" />

## Funciones de Callback

| Función                  | Descripción                                                                                   |
|--------------------------|-----------------------------------------------------------------------------------------------|
| `onTokenURI`             | Función de callback que retorna la URI del token para los tokens ERC-721.                     |
| `updateMetadataERC721`   | Función de callback que actualiza los metadatos para un lote de tokens ERC-721.                |

## Funciones de Fallback

| Función                     | Descripción                                                                                   |
|-----------------------------|-----------------------------------------------------------------------------------------------|
| `uploadMetadata`            | Permite cargar metadatos. Requiere el rol de minter.                                          |
| `setBaseURI`                | Establece la URI base para los metadatos de los tokens. Requiere el rol de administrador.     |
| `getAllMetadataBatches`     | Recupera todos los lotes de metadatos cargados hasta ahora.                                   |
| `nextTokenIdToMint`         | Retorna el siguiente ID de token que se va a acuñar.                                          |
| `getBatchId`                | Retorna el ID del lote para un token específico.                                               |
| `getBatchRange`             | Retorna el rango de IDs de tokens para un lote de metadatos específico.                       |
