# ClaimableERC721

## Descripción

El contrato `ClaimableERC721` es un módulo que proporciona funcionalidad de minting basada en reclamación para tokens ERC-721. Incluye características para establecer condiciones de reclamación, validar reclamaciones mediante pruebas Merkle, y gestionar solicitudes de reclamación basadas en firmas. Este contrato asegura procesos de minting seguros y controlados con la posibilidad de integrar ventas primarias.

## Funciones de Callback

| Función                         | Descripción                                                                                                    |
|----------------------------------|----------------------------------------------------------------------------------------------------------------|
| `beforeMintERC721`               | Función de callback que verifica las condiciones de reclamación antes de hacer el minting de tokens ERC-721.    |
| `beforeMintWithSignatureERC721`  | Función de callback que verifica una solicitud de reclamación con una firma y procesa el minting según corresponda. |

## Funciones de Fallback

| Función                              | Descripción                                                                                  |
|---------------------------------------|----------------------------------------------------------------------------------------------|
| `getSaleConfig`                       | Retorna la configuración de venta para un token, incluyendo el destinatario de la venta primaria.|
| `setSaleConfig`                       | Establece la configuración de venta para un token, incluyendo el destinatario de la venta primaria. Requiere el rol de administrador. |
| `getClaimCondition`                   | Retorna la condición de reclamación para un ID de token específico.                          |
| `setClaimCondition`                   | Establece la condición de reclamación para un ID de token específico. Requiere el rol de minter. |

## Interfaces Requeridas

| Interface  | ID          |
|------------|-------------|
| `ERC-721`  | `0x80ac58cd`|

## Callback de Instalación

Este módulo utiliza un callback de instalación para configurar los parámetros iniciales, como el destinatario de la venta primaria.
