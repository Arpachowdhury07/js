# MintableERC20

## Descripción

El contrato `MintableERC20` es un módulo que proporciona funcionalidades de minting para tokens ERC-20. Este módulo incluye características como minting con y sin firmas, y configuración para los receptores de ventas primarias.

<GithubButtonLink href="https://github.com/thirdweb-dev/modular-contracts/blob/dev/src/module/token/minting/MintableERC20.sol" />

## Funciones de Callback

| Función                          | Descripción                                                                                                     |
|-----------------------------------|-----------------------------------------------------------------------------------------------------------------|
| `beforeMintERC20`                 | Función de callback que verifica la autorización de mint antes de realizar el mint de tokens ERC-20.           |
| `beforeMintWithSignatureERC20`    | Función de callback que verifica una solicitud de mint con una firma y procesa el mint según corresponda.        |

## Funciones de Fallback

| Función                 | Descripción                                                              |
|--------------------------|--------------------------------------------------------------------------|
| `getSaleConfig`          | Devuelve la configuración de la venta para un token, incluyendo el receptor de la venta primaria. |
| `setSaleConfig`          | Establece la configuración de la venta para un token, incluyendo el receptor de la venta primaria. Requiere rol de administrador.       |

## Interfaces Requeridas

| Interfaz  | ID          |
|------------|-------------|
| `ERC-20`   | `0x36372b07`|

## Callback de Instalación

Este módulo usa un callback de instalación para configurar los parámetros iniciales, como el receptor de la venta primaria.
