# CreatorTokenERC20

## Descripción

El contrato `CreatorTokenERC20` es un módulo que se basa en el estándar ERC-20 para ofrecer funcionalidad adicional orientada a los creadores de tokens. Este módulo incluye características como la validación de transferencias para asegurar que solo las transferencias aprobadas sean procesadas, añadiendo una capa de control y seguridad a las transferencias de tokens.

## Funciones de Callback

| Función               | Descripción                                                                                       |
|-----------------------|---------------------------------------------------------------------------------------------------|
| `beforeTransferERC20` | Función de callback para las funciones `ERC20.transferFrom` y `ERC20.safeTransferFrom`. Ejecuta acciones antes de las transferencias de tokens, incluyendo la validación de la transferencia por parte del `transferValidator`. |

## Funciones de Fallback

| Función                          | Descripción                                                                                           |
|-----------------------------------|-------------------------------------------------------------------------------------------------------|
| `getTransferValidator`            | Retorna la dirección del contrato `transferValidator` para este contrato de tokens.                   |
| `getTransferValidationFunction`   | Retorna el selector de la función y el estado de vista para la función de validación del `transferValidator`. |
| `setTransferValidator`            | Establece la dirección del `transferValidator`. Requiere el rol de administrador.                     |
