# ClaimableERC20

## Descripción

El contrato `ClaimableERC20` es un módulo que proporciona funcionalidad de minting basado en reclamos para tokens ERC-20. Incluye características para configurar condiciones de reclamo, validar reclamos mediante pruebas Merkle y manejar solicitudes de reclamo basadas en firma. Este contrato garantiza procesos de minting seguros y controlados con la posibilidad de integrar ventas primarias.

## Funciones de Callback

| Función                         | Descripción                                                                                                    |
|----------------------------------|----------------------------------------------------------------------------------------------------------------|
| `beforeMintERC20`                | Función de callback que verifica las condiciones de reclamo antes de hacer el mint de tokens ERC-20.          |
| `beforeMintWithSignatureERC20`   | Función de callback que verifica una solicitud de reclamo con una firma y procesa el mint según corresponda.    |

## Funciones de Fallback

| Función                              | Descripción                                                                                  |
|---------------------------------------|----------------------------------------------------------------------------------------------|
| `getSaleConfig`                       | Devuelve la configuración de la venta para un token, incluyendo el receptor de la venta primaria. |
| `setSaleConfig`                       | Establece la configuración de la venta para un token, incluyendo el receptor de la venta primaria. Requiere rol de administrador. |
| `getClaimCondition`                   | Devuelve la condición de reclamo para un token específico.                                     |
| `setClaimCondition`                   | Establece la condición de reclamo para un token específico. Requiere rol de minteador.        |

## Callback de Instalación

Este módulo usa un callback de instalación para configurar los parámetros iniciales, como el receptor de la venta primaria.
