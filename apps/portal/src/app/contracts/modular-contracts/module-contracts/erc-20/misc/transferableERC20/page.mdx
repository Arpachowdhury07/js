# Transferable ERC-20

## Descripción

El contrato `TransferableERC20` es un módulo que proporciona funcionalidad de control de transferencias para tokens ERC-20. Este contrato permite habilitar o deshabilitar transferencias de tokens globalmente o para direcciones específicas, añadiendo una capa extra de control sobre cómo se pueden mover los tokens.

<GithubButtonLink href="https://github.com/thirdweb-dev/modular-contracts/blob/dev/src/module/token/transferable/TransferableERC20.sol" />

## Funciones de Callback

| Función               | Descripción                                                             |
|-----------------------|-------------------------------------------------------------------------|
| `beforeTransferERC20` | Función de callback para `ERC20.transfer`. Asegura que las transferencias estén habilitadas antes de permitirlas. |

## Funciones de Fallback

| Función                 | Descripción                                                                 |
|-------------------------|-----------------------------------------------------------------------------|
| `isTransferEnabled`     | Retorna si las transferencias están habilitadas globalmente para el token.   |
| `isTransferEnabledFor`  | Retorna si las transferencias están habilitadas para una dirección específica. |
| `setTransferable`       | Establece el estado de transferibilidad para el token globalmente. Requiere el rol de administrador. |
| `setTransferableFor`    | Establece el estado de transferibilidad para una dirección específica. Requiere el rol de administrador. |
