# ClaimableERC1155

## Descripción

El contrato `ClaimableERC1155` es un módulo que proporciona funcionalidad de acuñación basada en reclamaciones para tokens ERC-1155. Incluye características para establecer condiciones de reclamación, validar reclamaciones mediante pruebas Merkle y manejar solicitudes de reclamación basadas en firmas. Este contrato garantiza procesos de acuñación seguros y controlados con la posibilidad de integrar ventas primarias.

## Funciones de Callback

| Función                          | Descripción                                                                                                    |
|-----------------------------------|----------------------------------------------------------------------------------------------------------------|
| `beforeMintERC1155`               | Función de callback que verifica las condiciones de reclamación antes de acuñar tokens ERC-1155.                |
| `beforeMintWithSignatureERC1155`  | Función de callback que verifica una solicitud de reclamación con una firma y procesa la acuñación en consecuencia. |

## Funciones de Fallback

| Función                              | Descripción                                                              |
|---------------------------------------|--------------------------------------------------------------------------|
| `getSaleConfig`                       | Devuelve la configuración de la venta para un token, incluyendo el receptor de la venta primaria. |
| `setSaleConfig`                       | Establece la configuración de la venta para un token, incluyendo el receptor de la venta primaria. Requiere rol de administrador. |
| `getClaimConditionByTokenId`          | Devuelve las condiciones de reclamación para un ID de token específico.   |
| `setClaimConditionByTokenId`          | Establece las condiciones de reclamación para un ID de token específico. Requiere rol de minter.  |

## Interfaces Requeridas

## Callback de Instalación

Este módulo utiliza un callback de instalación para configurar los parámetros iniciales como el receptor de la venta primaria.
