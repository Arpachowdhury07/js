# Transferable ERC-1155

## Descripción

El contrato `TransferableERC1155` es un módulo que proporciona funcionalidad de control de transferencias para tokens ERC-1155. Este contrato permite habilitar o deshabilitar las transferencias de tokens a nivel global o para direcciones específicas, añadiendo una capa extra de control sobre cómo se pueden mover los tokens.

<GithubButtonLink href="https://github.com/thirdweb-dev/modular-contracts/blob/dev/src/module/token/transferable/TransferableERC1155.sol" />

## Funciones de Callback

| Función                   | Descripción                                                                                        |
|----------------------------|----------------------------------------------------------------------------------------------------|
| `beforeTransferERC1155`    | Función de callback para `ERC1155.safeTransferFrom`. Asegura que las transferencias estén habilitadas antes de permitirlas. |
| `beforeBatchTransferERC1155`| Función de callback para `ERC1155.safeBatchTransferFrom`. Asegura que las transferencias en lote estén habilitadas antes de permitirlas. |

## Funciones de Fallback

| Función                | Descripción                                                                             |
|-------------------------|-----------------------------------------------------------------------------------------|
| `isTransferEnabled`     | Devuelve si las transferencias están habilitadas globalmente para el token.            |
| `isTransferEnabledFor`  | Devuelve si las transferencias están habilitadas para una dirección específica.        |
| `setTransferable`       | Establece el estado de transferibilidad para el token a nivel global. Requiere rol de administrador. |
| `setTransferableFor`    | Establece el estado de transferibilidad para una dirección específica. Requiere rol de administrador. |
