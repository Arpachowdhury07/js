# SequentialTokenIdERC1155

## Descripción

El contrato `SequentialTokenIdERC1155` es un módulo que proporciona generación secuencial de IDs para tokens ERC-1155. Este módulo asegura que cada nuevo token acuñado reciba un ID único y en incremento, evitando colisiones de IDs y simplificando el proceso de acuñación de tokens.

<GithubButtonLink href="https://github.com/thirdweb-dev/modular-contracts/blob/dev/src/module/token/tokenId/SequentialTokenIdERC1155.sol" />

## Funciones de Callback

| Función                | Descripción                                                                       |
|-------------------------|-----------------------------------------------------------------------------------|
| `updateTokenIdERC1155`  | Función de callback para actualizar el ID del token para los tokens ERC-1155. Genera un ID secuencial si el ID proporcionado es el valor máximo y devuelve el siguiente ID. |

## Funciones de Fallback

| Función                | Descripción                                                                |
|-------------------------|----------------------------------------------------------------------------|
| `getNextTokenId`        | Devuelve el siguiente ID de token a ser usado para acuñar.                |
