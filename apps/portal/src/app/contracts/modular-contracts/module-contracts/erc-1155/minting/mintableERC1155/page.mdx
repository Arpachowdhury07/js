# MintableERC1155

## Descripción

El contrato `MintableERC1155` es un módulo que proporciona funcionalidades de acuñación para tokens ERC-1155. Este módulo incluye características como acuñación con y sin firmas, y configuración para los receptores de ventas primarias.

<GithubButtonLink href="https://github.com/thirdweb-dev/modular-contracts/blob/dev/src/module/token/minting/MintableERC1155.sol" />

## Funciones de Callback

| Función                          | Descripción                                                                                                    |
|-----------------------------------|----------------------------------------------------------------------------------------------------------------|
| `beforeMintERC1155`               | Función de callback que verifica la autorización para acuñar antes de acuñar los tokens ERC-1155.               |
| `beforeMintWithSignatureERC1155`  | Función de callback que verifica una solicitud de acuñación con una firma y procesa la acuñación en consecuencia. |

## Funciones de Fallback

| Función                 | Descripción                                                              |
|--------------------------|--------------------------------------------------------------------------|
| `getSaleConfig`          | Devuelve la configuración de la venta para un token, incluyendo el receptor de la venta primaria. |
| `setSaleConfig`          | Establece la configuración de la venta para un token, incluyendo el receptor de la venta primaria. Requiere rol de administrador. |

## Interfaces Requeridas

| Interfaz  | ID          |
|-----------|-------------|
| `ERC-1155` | `0xd9b67a26`|

## Callback de Instalación

Este módulo utiliza un callback de instalación para configurar los parámetros iniciales como el receptor de la venta primaria.
