# BatchMetadataERC1155

## Descripción

El contrato `BatchMetadataERC1155` es un módulo que proporciona gestión de metadatos por lotes para tokens ERC-1155. Este módulo se basa en las funcionalidades del contrato `BatchMetadataERC721`, adaptándolas al estándar ERC-1155. Incluye funcionalidades para actualizar y gestionar metadatos de manera eficiente en lotes.

<GithubButtonLink href="https://github.com/thirdweb-dev/modular-contracts/blob/dev/src/module/token/metadata/BatchMetadataERC1155.sol" />

## Funciones de Callback

| Función                   | Descripción                                                                                |
|----------------------------|--------------------------------------------------------------------------------------------|
| `onTokenURI`               | Función de callback que devuelve la URI del token para los tokens ERC-1155.               |
| `updateMetadataERC1155`    | Función de callback que actualiza los metadatos de un lote de tokens ERC-1155.              |

## Funciones de Fallback

| Función                     | Descripción                                                                               |
|------------------------------|-------------------------------------------------------------------------------------------|
| `uploadMetadata`             | Permite cargar metadatos. Requiere el rol de minter.                                      |
| `setBaseURI`                 | Establece la URI base para los metadatos del token. Requiere el rol de administrador.     |
| `getAllMetadataBatches`      | Recupera todos los lotes de metadatos cargados hasta el momento.                          |
| `nextTokenIdToMint`          | Devuelve el siguiente ID de token a ser minteado.                                         |
| `getBatchId`                 | Devuelve el ID de lote para un token ID específico.                                        |
| `getBatchRange`              | Devuelve el rango de IDs de token para un lote de metadatos específico.                   |
