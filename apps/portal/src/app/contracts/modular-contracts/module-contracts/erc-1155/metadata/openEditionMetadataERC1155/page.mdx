# OpenEditionMetadataERC1155

## Descripción

El contrato `OpenEditionMetadataERC1155` es un módulo que proporciona gestión de metadatos para tokens ERC-1155 de Edición Abierta. Este módulo amplía las funcionalidades del contrato `OpenEditionMetadataERC721`, adaptándolas al estándar ERC-1155.

<GithubButtonLink href="https://github.com/thirdweb-dev/modular-contracts/blob/dev/src/module/token/metadata/OpenEditionMetadataERC1155.sol" />

## Funciones de Callback

| Función          | Descripción                                                  |
|-------------------|--------------------------------------------------------------|
| `onTokenURI`      | Función de callback que devuelve la URI del token para los tokens ERC-1155. |

## Funciones de Fallback

| Función            | Descripción                                                                               |
|---------------------|-------------------------------------------------------------------------------------------|
| `setSharedMetadata` | Establece los metadatos compartidos para la colección de tokens ERC-1155. Requiere el rol de minter. |
